站长网专访：A5会客:解密逐浪CMS研发八种武器

作者:本站编辑 发布时间:2013-11-12 来源:本站原创 点击数: 分享网址

 

 

 

 

　一年前还在和你谈是HTC手机好、还是三星手机好、又或是带着客户到公司楼下的apple浦东旗舰道店挑选实物，今天他已经是三次奔赴中国毛笔之乡—文港镇潜心探寻一支属自己的好毛笔。

 

　　一年前还在和你谈是茶健康还是咖啡提神，今天他已经信奉“果珍李柰　菜重芥姜”①的朴素生活观。

 

　　一年前还在和你谈是画怡情还是书成人，今天已经成为专注严谨书法的瘦金体门人。

 

　　是什么让Zoomla!逐浪CMS掌门人“发哥”如此突变?

 

　　带着疑问，A5专访走进位于上海Zoomla!逐浪CMS软件研发中心，与发哥面对面，透析大数据与WEB开发的的未来。

 

 

 

　　关于研发—代码之夜与元旦不休的狼人传说

 

　　A5编辑问(以下简称问)：能否透露下个版本的Zoomla!逐浪CMS研发计划?

 

　　Zoomla!逐浪CMS发哥②答(以下简称答)：可以，Zoomla!逐浪软件的下个重要的版本目前定名为CMS2 X1.2，目前已经在紧密的内测中。

 

　　在Zoomla!逐浪CMS2 X1.1发布成功后，其中的集成微信应用、生成APP、访问设备识别、MIS系统支持等用户功能，深受广大高端客户的喜爱，大大鼓舞了我们在高端网站系统中的信心，从而推进新版本的研发。

 

　　新版中，我们再度加强了企业化的管理，包括改进的服务器兼容与支持，包括基于OS系统操作层面的站群和标签引用，使开发效率进一步提高，数据挖掘更加方便。

 

　　每年的元旦，Zoomla!逐浪CMS团队都会发布一个全新的版本，在时间与空间上都力求引导整个行业的发展，全体华夏互联人都会为这个技术理想而奋斗，哪怕放弃全世界人民都在休假的权利!

 

　　相信今年元旦的代码之夜将是一个精彩的技术之夜，因为全新的Zoomla!逐浪CMS2 X1.2正在等待发布。

 

 

 

　　关于架构—右灰线与钢构之区别

 

　　问：目前市面上CMS产品主要集中在dotNET和PHP两大阵营，二者的区别主要在哪里呢?

 

　　答：如同盖房子，往往先打基础。

 

　　在打基础前，大师傅会用右灰画一个基础线，画好了线，工人就好顺着线的痕迹开挖，由此开启万丈高楼平地起。

 

　　然而现代化的工艺不再采用传统的开挖，直接钢构与预筑板建设。

 

　　我们将前者往往比喻成PHP类的网站，如果你有足够高的技艺，可以不断的重构，PHP类的CMS往往只起了引子作用。

 

　　所以我们在PHP应用市场看到两个极端：一个极端是高端应用的典范，另一个极端则是千篇一律的仿站“源码改装”，毫无创新之处。

 

　　而如同Zoomla!逐浪CMS的dotNET产品定位，是做网站的钢构框架和技术之核，用户不需要去捉摸纯底层知识，却又能依托庞大的架构和其后台的云技术管道，不断的接入新的服务、开发新的用户功能、进行深层的数据挖掘，显然拥有更加优秀的扩展性。

 

　　当然，更高的产品定位对用户技术和知识层面也就有了更高的要求，这也是利弊共存的，好在Zoomla!逐浪CMS的定位就是只为开发者开放，专注于高端市场。

 

　　所以我们有客户送我们一首诗，虽然目前不够好，但我们还会持之以恒的努力：

 

　　建网站，选逐浪。

 

　　易上手，功能棒。

 

　　质量好，服务善。

 

　　高性能，最划算。

 

　　关于趋势—万变不离其宗的互联网基石

 

　　问：曾经有一段时间，业界普遍唱衰PC互联网将走向消亡，而移动互联网将成为主流，你如何看?

 

　　答：家用娱乐产品线与企业级产品线二者并不冲突，如果你认真的去研究，就会发现，任何移动互联网平台背后都是是依托着庞大的PC互联网，没有了这一环节则如同无源之水，显然不可能长流。

 

　　而企业级市场离不开个人市场的发展和创新，这就取决于一个公司的产品创新与服务水平了。

 

　　Zoomla!逐浪CMS产品在研发和服务全过程中，始终恪守“服务在现场”的理念，无论是北国雪飘、还是西藏高原、或是南海之滨，都留下了Zoomla!逐浪CMS程序员的脚印，这就是“脚比路更长”的企业精神体现。

 

　　我们还可以透露一点，与一些做推广竞价的企业总是希望客户群体无限扩大不同，Zoomla!逐浪CMS认为客户的服务基数是有限的，我们只能服务于固定数量的客户，才能保证服务品质，因而我们的销售工程师在接洽客户过程中，并不是无限制的接受，而是有原则、站在与客户一致的角度去规划、构思，并可以将其纳入庞大的逐浪CMS开发者社区。

 

 

 

　　关于客户—突然十年前的老客户找你，那是一种幸福

 

　　问：什么事使你感到幸福?

 

　　答：同事取得新成就、同行取得新发展，都能使我感到幸福。

 

　　比如突然某一天，一个10年前的老客户来找你，说他的网站在发展中遇到新问题，需要一起探讨，这也是一种幸福。

 

　　又比如千里迢迢的去西北农村，为当时的站长解决现场问题，让这个地区的互联网门户从无到有实现历史的跨越，同样也是一种幸福。

 

　　作为稳固结构的逐浪CMS，有着强大的适应性、扩展性和鲁棒性③，往往客户很多年以后还需要扩展升级或二次开发，而我们乐于为客户提供成长的解决方案。可以毫不夸张的说，祖国的每一个省份都留下了逐浪CMS人的足迹，即使今天客户遍布中国，并拥有一批稳定的海外客户，但我们的服务准则依然不变。

 

　　我们的销售规则中，严格杜绝以简单的“年度服务费”来销售产品，而是告诉客户，我们的成长是有赖于客户的成长和新增需求来发展，从而将客户的成长与自己的成长捆绑在这一起，做负责任的企业。

 

 

 

　　关于产品—低调务实做产品

 

　　问：能为使用逐浪CMS的新用户和广大的A5站长群体提供一些建议吗?

 

　　答：只能说分享一些心得。

 

　　首先是专注，比如分类信息平台，当年各类产品的分类信息平台何其多，存活下来的没有几家。即使是赶集网的战略也是摇摆不定，唯独58同城能够做出来了。这回好了，58同城上市，我听到身边的朋友好多说也要再次进军分类信息平台了!

 

　　其次是服务，好多站长关注“流量”，当为了流量而做流量时，整个流量指数就毫无意义了。

 

　　再次是坚持，不恒其德、或承其羞。世界上没有不劳而获的东西。

 

　　古语：试玉要烧三日满，辨材须待七年期。

 

　　可见努力的重要性，而我们现在年轻人，总希望两三年就包装出一个产品能够融资上市，这显然是不符实际的。

 

　　鄱阳湖畔千钧立业，

 

　　井冈山下扬帆启航，

 

　　黄浦江上谱就新篇，

 

　　东方明珠下逐浪来。

 

　　2009年我和同事扛着电脑主机、打印机来到上海浦东，开创了上海逐浪CMS软件科技有限公司，与南昌公司一体推进软件体系的研发，这其中凝聚了全公司的人们的心血。

 

　　即使到今天，我们依然坚持自己的原则：

 

　　第一只专注技术，纯技术无销售，虽然销售可以让我们的收入夸大，但我们要看到更远的未来。

 

　　第二专注于企业市场，我们公司有一个很著名的“99万定律”，意即“100万的政府订单只赚一万”，坚决杜绝请吃喝送所带来的“99万”。虽然同事们没有了花天酒地、或是吹牛扯泡的机会，但是感受到的是真真正正务实的做产品，客户也愿与我们这样的公司打交道。

 

　　第三是“小公司”定位，我们固执的认为：100个人的公司如果99个是业务员，那还不如一个人的小公司，而我们就是那个技术型企业，做让客户放心、让员工舒心的沪上研发企业。

 

 

　　上图：逐浪CMS发哥与外藉专家合影

 

 

 

　　关于书法—专治“小学”严谨求索

 

　　问：听说某人夸发哥你是“站长中的书法家、书法家中的站长”，莫非发哥也虚伪了，玩起了所谓的书法?

 

　　答：哎哟，我可不敢玷污书法这两个大字，我称之为小学而已，所谓小学是古人将文字学、音律学、训诂学三者合为一体的总称。

 

　　在我们“程序猿”脑海中，所谓书法家，好象是要在裸体模特上涂画、又或是能作出没人识得的“写意”作品的“大方之家”。

 

　　而我试图去恢复华夏文明写作的规矩之道，从标准之楷入手、从辉煌宋之瘦金起步，力求不正不写、不方不作。

 

　　我平常是不敢在办公室写作的，但无赖好端端的一块两米长的上好婺源歙砚，总是被福建客户和小伙伴们认为这是一个茶具，而我又实在不敢享受人生钻研茶道(我甚至不认为有此一道)，只好在工作之余拿起毛笔写几个字，丑得不行。

 

　　不过依然要顺便鄙视一下那些连楷书都写不好的所谓书法家或是国学家们，其实是沽名钓誉、钻营私利。

 

　　概言而之，学书法也是为了提升Zoomla!逐浪CMS的含金量，我们要做的是最懂中国文化和高深思想的软件产品，就必须拥有人文气质，否则只能是徒有其表的“土豪”，我们不敢说中国站长圈的书法家，但敢说是中国CMS行业中最懂中国文化和客户需求的CMS产品!

 

　　--只有懂客户需求，才能提供最合适的产品和实施服务，并提供一以贯之的服务和增值。

 

　　至于虚伪两个字，我倒是承认，年轻时过多的爱慕虚荣，而没有更好的去关怀好同事，在产品上也有很多遗憾，以后将全力的收起华丽的羽毛，去专注做好企业经营。

 

　　企业不在人多，在于产品创新。

 

　　企业不在产值飘高，而在于利润可人。

 

　　见之于逐浪CMS，则是力争做中国高端CMS的第一品牌。

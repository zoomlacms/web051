《生意通》苦水里泡出的企业家

作者:《生意通》苦水里泡出的企业家 发布时间: 来源:本站原创 点击数: 分享网址


     ——江西最大的非电信虚拟主机服务商；
     ——江西唯一具有自主知识产权软件的网站服务商；
     ——江西最大的域名服务商；
     ——江西最专业的网站推广与服务商，其客户包括诸多政府以及大型外资企业；
     “打开全球网站排名系统，我们华夏互联是全省同类网站排名最高的，较第二名排名高出一大截”—坐在眼前的这位年轻小伙子如数家珍的向记者介绍着自己的业务体系，他就是我们本期创业故事的主人公、著名网站与电子商务服务商华夏互联(ww.hx008.cn)总经理、江西聚合实业有限公司掌门人汪照发先生。
    
     幼小丧父母 立志做强人
     汪照发，系江西婺源人，1980年出生，祖籍浙江衢州。
     他原本有着一个幸福美满的家庭，但是，这一切在刚出生不到一岁的时候就发生了改变，父亲当时是生产队长，在一次耕田劳中死于工伤。汪照发5岁时母亲改嫁他人，之后不到一年，母亲也因病撒手西去。从此，小小的照发只得跟着外祖父、外祖母一起生活，但是舅舅因为家里多了一个负担，居然与自己的二老断绝了关系，而且把父母驱逐出家门。从此，外祖父、外祖母与小照发三人相依为命，两位老人为了供他求学，吃尽了苦头。
     随着家里老人年纪的增大，汪照发一方面要照顾家中老人，一方面要继续修学。半工半读的生活，让他领悟到生活的艰辛苦，因此他学习非常努力。后来，他通过自己的努力，修完了浙江大学经济管理学课程。学成后，汪照发毅然选择了下海经商，在一家民营企业做起了企业策划与管理的工作，并以自己的勤奋、好学赢得了领导与同事的尊重，很快成为岗位上的技术能力与团队里的带头人。
     三年前提起汪照发，在江西传媒与企业界就小有名气，作为某外资地产企业的总策划、曾获上饶市企业文化建设一等奖（最高奖），后任省城某知名报社记者、编辑与策划负责人，原本是可算小资的白领生活，而当时我创办的华夏营销网（www.hx008.com）已经是中国最具知名的营销策划网站之一，在经济、传媒与文化界拥有一定的影响力。
     出于对事业与理想的执着，我于2003年注册成立了自己的企业（江西聚合实业有限公司），公司以专注运营计算机工程、网络化市场与软件开发，现已成为江西省内专业的集虚拟市场、网站程序开发与建设、电子商务运营的领先企业，旗下拥有华夏互联(www.hx008.cn)、华夏营销网（www.hx008.com）、华南学习网(www.i37.cn)三大平台，在省内外同行中享富盛名。
    
     从传媒到互联网 艰苦创业构建宏图
     准确定理，理性创业。
     谈起互联网，很多人都容易产生误区，或者认为网络创业是可有可无的虚幻产品，或者认为网络创业只要建设一个主页或者开一个网店就算是成功了，更有甚者，认为网站制作只要找一些开源代码进行稍微的改进就可以了。
     事实上，这些都是对电子商务的曲解。电子商务是时代进步的灵魂，而促进这个灵魂进步的则是技术研发的不断进步。为了保证华夏互联平台的进步，长期以来，我们把所有的收入都投入到硬件升级与产品研发中，确保了技术的领先，目前包括南昌军分区、江西省邮政、江阴热线、深圳迪斯尼等实力企业都成了我们的忠实客户。
    
     诚信经营，打造百年老店
     谈起这些年的创业经验，有两件事让我记忆犹新。
     一次是高安市一个政府项目网站建设成功，在我们自己的服务器上调试完全正常，而在对方的服务器上调试不成功，原因显然是对方的服务器有问题，根据合同规定，我们是不需要对此负责的。但本着对客户负责的态度，我们为此亲派了专家组亲赴高安，不收一分钱就帮客户解决了困难，并赢得了客户的高度认同，至今高安市的多个农业网站项目都是定点选择华夏互联建设的，在当地享有较高的声誉。
     还有一次，南昌一个大型政府机构在财务管理中出现误差，将一笔货款重复支付了两次，当我们的财务人员发现了后，请示我，问我是如何处理这笔款。当时我就毫不犹豫的指示：这笔款必须立即如数的归还客户。因为我们深知，诚信是企业的立业之本，虽然客户没有发现财务问题，但我们有责任与客户同舟共济，创造共赢之道。
    
《生意通》2007年第3期

MSN中国:2015江西互联网圈:专注研发的逐浪软件篇

作者:本站编辑 发布时间:2015-03-01 来源:本站原创 点击数: 分享网址

 

【转载自MSN中国】

 

　仆窃不逊，近自托于无能之辞，网罗天下放失旧闻，考之行事……亦欲以究天人之际，通古今之变，成一家之言。草创未就，惜其不成。仆诚已著此书，藏之名山，传之其人通邑大都，则仆偿前辱之责，虽万被戮，岂有悔哉!

　　—司马迁《报任安书》



　　当听朋友说，作为当今世界“吸金吐银”最为频繁、所有资本集中的软件和互联网领域，如果还有人为“著此书藏之名山传之其人”，那么这理想的火花必定闪耀在逐浪CMS软件研发团队的身上，编辑部的老师们是带着怀疑态度的。

　　而当我们走近他们时，一个个鲜活的面孔、蓬勃向上的员工风貌却不得不让我们相信眼前的事实。

　　

　　这是一个“潜伏”在互联网后台的软件研发团队，他们的CMS系统是无数个网站的核心，定位 “做软件公司的公司”，“无销售纯技术”模式别树一帜，以致10年来耕耘将中化集团、中电集团、南京军区以及诸多世界五百强企业的订单悉收囊中，远者连西藏政府、澳州、新西兰客户也纷纷通过外包平台找到他们，因为其旗下华夏互联与逐浪CMS两块金字招牌，代表了卓越WEB软件的金牌信誉与保障。

 

　　采访逐浪软件是困难的，因其总有加不完的班。

　　写尽一个人是困难的，因为这是一家只信奉“明天更精彩”的企业，并不认为当下工作有什么值得夸扬之处。

　　为此小编只得以其核心团队三个成员切入，以求剖解。

 

 

　　小末：介于傻姑与潮客间的经理

　　“我们不是一家新公司，已经在中国软件研发领域默默耘了10年的老牌企业，深谙成人之美与恒德恒心，专注底层、力求最终用户看到的产品不是我们，而是伙伴与集成商的LOGO” 。

　　—小末开场的第一句话道出了逐浪软件的生存哲学。

　　

　　不同于江南西道多山多河，作为公司少有平原上长大的姑娘，一来公司就做了六年，人们说她傻，她却笑道：一千多年前的先贤就会说“试玉要烧三日满，辨材须待七年期”，今天我做这一点算什么?!

　　在华夏互联，正是有一批为梦想而执着的程序员，他们甚至都不应称为普通的开发者，而是科学家、实践家，并通过程序实现了自己的理想。公司能积累人才是对员工充分的信任、并给予丰富的回报和积极的培养成长。

　　Nokia、三星、Lumia、surface、iPhone 5s、iPad、Kinder、Xbox One、Kinect For PC……当这些硬件全部出现在一个桌上时，不必惊讶，这就是被人称“傻姑”的小末桌上，因为要做各种硬件的适配，所以小末手上的IT设备永远是最新的，几乎每两个月就要换一部手机。

　　而她指着一个键盘说：诺，不要看它其貌不扬，却是正宗德国货樱花牌键盘，价格要3000多!!!我们团队也是坚守筚路蓝缕、唾面自干的精神低调而不失新锐。

　　而对于人才招聘，小末说我们很简单，一是工资很高，二是条件很好，三是设备很优。你看我们的客户机，跑的都是企业级的windows Sever2012R2系统，而家用级win7、win8以及xp在逐浪公司是站不住脚跟的，“我们就是要做到挥舞大刀却又细致入微”的专业职人。

 

 

　　婷婷：信奉“脚比路更长”的行者

　　“我们也在改变世界，不是做世界的随从，因为我们就是互联网的造梦师和建筑师。”

　　--没错，那就是婷婷，婷婷玉立、脱颖而出的研发经理。

　　

　　如果你看她是弱女子，那就错了，因为她正是著名的逐浪CMS响应式平台核心开发者之一，成就了国内首家基于响应式应用的网站内核产品。

　　婷婷说，在逐浪软件的开发团队中每一个人都具备自由发展的空间，公司定位是只专注于技术，甚至提出需要无数个“小矮人”组成不可攻破的斯巴达方阵，每个客户都力求服务到现场。

　　即使在江西省内，也包括江西日报社、江西商报、工信厅、国税等多个大型企业的软件和管理平台采用逐浪CMS架构，依托的正是我们强大技术实力和研发后续保障。

　　作为中国装修之乡出来的新秀，婷婷说：谁说当代武宁人只会做装修?软件研发与互联网我们同样在行!

　　关于职业生涯，婷婷表示自己看惯了好多年轻人不知天高地厚，温州深圳走一圈，然后赤条条的回老家的侯鸟般IT民工遭遇，自己将在这个大舞台上积累学习、沉淀自我、厚积薄发—时代永远只属于独立思维的行动者，而不是顺势无为的享受者们。



　　徐平：互联网幕后的建筑师

　　“再小的个体也应该拥有独立数据，尤其是中国本土企业更需要有自主软件支持，我们一直为此而奋斗。”

　　--徐平厚重镜片背后的眼神坚定而又不失儒雅，仿佛是作战的参谋和科研的教授，承担起整个逐浪软件的架构研发。

　　

　　每天有无数个网站诞生，每天又有无数个网页被请求，而构成这一切的最小单元就是CMS系统。每一套CMS都有其鲜明的生命力，这些别人看不懂的代码，在徐平的眼里却是生动活跃的。

　　让徐平最难以忘记的是03年与金蝶软件的一个接口协议开发，作为传统软件的领导者，金蝶选择逐浪CMS合作看重的是在WEB领域的实力，然而在整个开发中软件的交互和用户体验的改进没有想象的那么简单，虽然事后项目得到了完美实施，但逐浪软件公司随后主动为客户提供了三个版本的更新，以求加固，赢得了客户的高度赞赏。

　　虽然目前已经有很多公司采用了逐浪CMS进行二次开发，但是我们始终恪守只做核心、不导流量的原则，绝不向产品中放入插件和特殊管道，免费与收费版功能一致开放，甚至连后台LOGO都可以自由修改，并提供强有力的保障和技术支持，要让最终的软件产品看不到逐浪的影子才叫成功，真“成人之美”古风矣。

　　即使新一代的逐浪CMS2 X1.5发布，带有响应式和移动支持，并能与虚拟化 云系统接合，也一样的恪守了免费开放原则，值得赞誉。

　　做最笨的人—这是华夏互联团队口头禅。

　　所以，这么一群人放弃了在南昌的安逸生活，积极开拓上海、北京和海外市场。

　　所以，这么一群人定位“只做技术”，靠技术养技术，并将产品开放免费下载，成为江西省内唯一一家提供软件下载的ISV(独立软件研发厂商)。



　　正如他们的企业歌曲《大江》中所唱：

　　鄱阳湖畔激情创业，

　　井冈山下扬帆启航，

　　黄浦江上谱就新篇，

　　东方明珠下逐浪来。

　　江西软件精英弄潮上海正当时!~

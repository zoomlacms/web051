CNET中国·PChome.net:国内首家集成SNS的CMS厂商诞生

作者:本站编辑 发布时间: 来源:本站原创 点击数: 分享网址

编者按：2008年网络上刮起了一股激烈的SNS旋风，校内网、开心网、51.com等频频出新，国内最大的即时通讯公司腾讯也跃跃欲试，康盛创想也实时推出UCenter Home，其它论坛厂商纷纷跟进。

SNS即是社区关系网络，基于社区关系网络的平台构建，使网站拥有了更好的人机交互体制，并能够快速的通过“小众移植”，建立起非常稳固、可信的人气网络，这也是其赢得诸多网友亲睐的原因。

2009年元月1日，业界知名网站内核系统开发厂商—Zoomla!逐浪CMS领先发布了其年度大作—逐浪CMS2.0版，作为国内CMS厂商的领导者，Zoomla!逐浪CMS团队始终致力于为用户提供最佳产品体验与最优一体化网站架构解决方案，在其推出的2.0版本中融合了闭多其SNS功能，为广大站长带来全新的福音，成为国内首家集成SNS模块的CMS厂商，引领业界成长。

让我们一同走进Zoomla!逐浪CMS2.0的社区关系化网络世界。

专注开发成就品质

Zoomla!逐浪CMS软件定位于面向大型行业网站、垂直门户和诸多中小企业网站的一体化软件解决方案服务商，其SNS功能（Zoomla!逐浪SNS）早在2008年初就已发布，作为逐浪软件旗下的一个单独产品线发展，先后为包括布兜网、我要队伍、搜呗网络等平台提供了良好的专业社区化网络基础内核架构服务。

成熟的开发、长期的研究、高端的平台，决定了逐浪SNS系统的卓越功能，现已成为国内.NET平台下社区关系化网络软件的重要一员。

此次将SNS模块合并，将其列入逐浪CMS标准软件应用，一方面是对于产品质量的信心十足—经过一年的研发，其产品线已经达到完全可与CMS系统融合应用的境界；另一方面是基于对用户负责、回馈广大逐浪CMS软件用户的产品理念，为用户提供最佳的产品体验与功能应用。

正如2008年11月26日，逐浪软件三剑客应邀参与CSDN举办的SD大会时，CSDN网站CEO所问：“逐浪软件产品家族如何定义与发展？”一样，逐浪软件的创始人发哥的回答是：“一切以用户体验为首，当我们的产品到了足够成熟，而用户又有需求的时候，我们就会豪不犹豫的投入研发，而不是简单的为了商业化软件、放弃性能、“伪”开源而存在。”

当企业网站遭遇SNS—激活传统网站升级

“真正缺少用户体验与互动的是广大的企业和中小垂直网站，而只有我们的企业应用才是SNS的赢利所在”—这是一个逐浪CMS用户的心声，虽然不代表普遍的精英观点，但透露出了传统企业对于社区互动与关系网络的认知、需求。

有一组数据可以说明当前中小网站以及传统门户所存在的问题：

大约有80%的中小门户网站会员数据是孤立的，只能单向的接受网站方的广告邮件或电话垂询；

大约有90%的企业网站没有会员交互功能，只能单向的以“个人主页”形式向访客传递信息，从而造成了一大批专为搜索引擎而生存的网站，而非面向经营与战略而生存的网站；

大约有50%的网站各个模块间是互相孤立的，网站间依然存在诸多的信息孤岛……

而根据华夏互联多年来从事IDC领域的研究数据表明，多数访客对于网站互动性表现失望的模块主要体现在主站信息及商务模块，而不是论坛、博客等本身就是互动的节点。

Zoomla!逐浪CMS2.0版本中集成SNS功能，正是为了解决传统企业网站的缺陷，向无障碍网站交互方向发展。

Zoomla!逐浪CMS2.0中引入SNS功能，不仅是仅是为用户提供一个可以玩、学、游、乐的交互平台，更是致力于提供一个开放的计算平台，标准化的API、开放的计算，把冗余用户化为真正的访问流量，把访问流量化为真正的商业机遇。

娱乐、社区、交友、工作统统搞定—让用户粘住你的网站

长期以来，有不少的站长向逐浪软件项目组反馈：网站建起来了，但不能粘住用户，访客往往是“壮士一去不复返”。

引入SNS模块后，网站可以拥有丰富的互动。

抢车、小屋、蒙面聊、日志、点评、空间、主页等功能，人的娱乐性在这里得到了充分体现，为虚拟网络世界加入更多的现实元素。

以抢车为例，在游戏中用户可以购买自己喜欢的车辆，拥有一辆好车对于我们每一个人都不再是遥不可及的梦想。我们可以通过抢车位赚取更多的钱，在购买更好的车，不管你是小孩还是大人都是平等的，都可以很快实现自己的爱车梦。同时我们还可以自己添加更多的游戏，如虚拟朋友：结交一个虚拟的朋友，平时好好照顾他，无聊或郁闷的时候，可以来找你的朋友，让他陪陪你，逗你开心；抢床位：睡觉就可以赚钱喽！霸占好友的床位，睡觉，赚钱，买房子...可以把好友买来当瞌睡虫替你睡觉，还可以在别人的床上放蟑螂老鼠整人哦。用户乐此不彼，粘住用户，基于.NET平台开发的强大内核，满足高性能的应用。

人的社会性在这里也一样可以体现，每个正常人都不愿意与外界隔绝，而对于在网络上生存已经变成另一种不可或缺的技能。在这个虚拟的世界里我们同样也会交到更多的朋友，可以做更多我们在现实生活中无法做到的事，我们还可以参与和发起不同的活动，有些还能在现实得到实现，如同城缘、同生缘等等。也可以搜索好友的群组，加入不同的群组。更可以创建自己的群组，体验做“大哥”的感觉。

这就是个性化的功能，根据人的行为和生活方式组织不同的信息。我们可以发表日记，可以上传照片，甚至可以根据自己的好恶进行一些应用设置。我们在网络的信息海洋里游弋，有时会感觉到无所适从，因为大量的信息对于我们来说反而成为一种负担，我们在这里完全可以得到更多更实用的信息。这些信息对于我们来说绝对是有用的。

便捷应用贴近用户

普通的企业离SNS是不是很远呢？SNS难道只能用来玩？在C#和.net领域为何没有顶尖的SNS开发厂商？

答案是否定的，不管SNS的门槛有多高，不管前进道路上有多么艰辛，总有这么一群默默无闻的人在努力，不抛弃，不放弃是他们对自己的解嘲。其实我们每一个企业在电子商务的实现过程里都能找到自己的一片天地，对于每一个企业来说，去羡慕开心网是没有任何意义的，同样，我们也并不指望自己也能发展的像开心网那样，能自力更生开发那么多的应用，并能迅速取的那么大的成绩。

通过逐浪CMS的坚固内核和商务应用体系，逐浪SNS功能将得到极大的提升，用户可以快速建立一个与站点内核和经营主体紧密关联的社区门户，决胜于信息时代。

应用案例

图书馆多维互动体系：N大学的图书馆网站始建于三年前，虽然有专业的管理员和不懈的推广，然而广大的会员都不愿意登陆网站，即使登陆也仅限于检索信息，还会招致不少用户抱怨功能差，访客满意度极差。

导入逐浪SNS功能后，用户可以享受一站通的服务，简单三步注册即可拥有个人空间，用户在检索信息的时候可以快速的加入标签，可以随时发表书评，可以随时方便的将个人信息进行记录。管理员可以根据需求建立起简单的兴趣群组来讨论，用户可以随便更换“马夹”聊天。

文献网站会员：华南学习网是一个为用户提供学习资讯服务的网站，会员拥有近十万个，然而随着100E等平台的兴起，用户可以简单的在这些“搜索”型大站找到信息，用户流失很大。华南学习网也尝试过建立同样的“搜索”平台，可结果并不理想。

后为网站导入了SNS功能，反其道而行提出“个性学习我作主”的理念，致力于为用户提供真实的社区网络，建立起了基于各个分校的紧密交际平台，赢得了广大用户的芳心，重回区域门户之门。

企业门户员工关系网络：晶陵科技是一个国有大型软件开发集团，由于开发企业的特征，企业内网并不能对外公开，内部网络也不能访问外部局域网，造成员工怨声载道。

晶陵科技采购了逐浪CMS软件后，企业快速构建了一个基于WEB2.0理念的网站，用户可以通过内网快速访问，同时具备所有的时尚应用，抢车位、建空间、发书评、建族群、蒙面聊……员工间还可以建立兴趣群组，员工间交流也更加紧密了，企业文化建设上了一个新台阶。

[人物专访]严莎:行走在代码世界

作者:本站编辑 发布时间:2011-08-30 来源:本站原创 点击数: 分享网址

2011年8月19日《江西商报》第1615期B1版专题报道：

她是阳光灿烂的妙龄女孩，
她是默默无闻的卓越程序员，
她是鼎鼎有名的华夏互联美女CTO。

是什么力量，
成就了一位颇具远见卓识、敢打敢拼、外柔内刚的奇女子？

“兴趣”严莎微微一笑，直接抛出了答案……

 

她是阳光灿烂的妙龄少女，她是默默无闻的电脑程序员，她是鼎鼎有名的华夏互联美女CTO。她，就是那个名叫严莎的女孩。

是什么力量，成就了一位颇具远见卓识、敢打敢拼、外柔内刚的奇女子？

“兴趣。”严莎微微一笑，直接抛出了答案。

 

孱弱的体质，细细的声音，淡定的眼神，无论如何都很难与“半夜还在敲代码、电脑前一坐就是大半天、要从千万个代码中找出问题的那一句、从无数的BUG中找到最臭那条虫”的程序员相联系起来。

没错！她是阳光灿烂的妙龄少女，她是默默无闻的电脑程序员，她是鼎鼎有名的华夏互联美女CTO。她，是那个名叫严莎的女孩。

 

热忱是一盏灯

严莎的办公桌上没有过多女性化的装饰，如果不从细节上过多考量，根本无法让人感觉出那属于一位女技术总监。而严莎，作为华夏互联高管，这位在南昌计算机技术圈里极负盛名的女性，举止大方，成熟稳重，睿智而不失优雅。

整个采访过程中，严莎始终面带微笑，她思维开阔跳跃，言语简单朴实，一如电脑里闪动的编码。是什么力量，让一位如水的姑娘对冰冷的电脑软件有着如此强大的好奇心？又是什么力量成就了如此一位远见卓识、敢打敢拼、外柔内刚的奇女子？

“总的来说事实上就是一点，兴趣。”善于总结的严莎微微一笑，直接抛出了答案。“大学里学的就是软件，因为我觉得它富有挑战性，加上它对我来说有着一种神秘的吸引力。”

都说程序员是寂寞的，程序员的差事是苦差事，这一点对于当时的严莎来说，并不是没有想过。“各行各业都有它们的难处，我选择了这个行业就是因为它的神秘等着我探索，花点时间是应该的。只要我们对它有兴趣，任何困难我们都应尽自己所能去克服，去解决。”

“学不到新知识的原地踏步，让你感觉很空虚，甚至恐慌。”回忆起刚入职的时光，严莎露出焦虑的神态，停顿片刻说：“那个时候，大多数人都会想，我将来到底要做什么？其实我全部的需求不过是知识的积累。”

每个人都会经历种种忧郁和踌躇，一句平平常常的话，暗地里隐藏了太多的思想交锋。而这场交锋中，她做了从容自然的选择，时间也印证了她的正确。成长路上，年轻的严莎始终保持着理性的认识：耐心一些，低调一些，内敛一些。

参加工作之后，在看似枯燥的代码世界来去自如，严莎也没有太多烦扰，依然单纯如水，唯一坚定的是对程序的喜爱。“女人大多感性，那就借助这种力量，去工作中体验让你心动、感兴趣的部分。”

时间可以让人年华老去，也可以发生很多的偶然。那时候的严莎也没有想过，当初只是下意识的选择就业，谁知在编程世界畅游了几年之后，竟然转身一变，担任起华夏互联CTO重任。“面对空白区，要拿出最初的热情，并且敢于提出自己的意见，去闯去碰去沟通，随着时间的发展，积累新的创意。”

可能有太多的女性抱怨自己的职业不够时髦，工作内容单一、枯燥乏味，还要面对巨大的工作压力。那么严莎是如何保持对工作的热情，让自己全情投入的呢？严莎没有直接回答这个问题，她将目光转向了办公桌上的那台电脑，说：“当你完成了一组复杂的代码，看到程序run起来，你希望得到的东西就会呈现在你的面前。”一直语气平和的严莎，说道这里像孩子一样笑起来，“这让你觉得自己很聪明！做过程序员的人都会有这个体会，我喜欢这种代码流于指尖的快感和成就感。”

而事实上，她自己就是跳跃在键盘上的美丽精灵。因为对工作单纯执着的热爱，严莎的思片刻都不曾老去。这股热忱，就像一盏明灯，照亮她的职场人生路。

 

微笑的战士

作为江西省内为数不多的以底层软件研发为中心的企业，华夏互联早在2006年就启动了CMS网站内容管理系统研发，并在上海浦东建立全资研发中心，它就像是一面旗帜，引领者行业大军，昂首阔步行走在前方。

高质量的队伍无疑是卧虎藏龙的，而严莎作为这支队伍里面的一员，她又是怎样区别于旁人，成为最耀眼的一颗？

用同事的话来说：“她有着阳光般的心态。”而在严莎自己看来，这一切都平淡无奇，“我只是做好自己的工作而已，按时完成工作任务，做项目的时候注重细节，设身处地为客户着想，这都是本职工作。”都说程序员是单纯的，严莎便是典型的例子。对于大多数人，他们对最简单最基本的事情往往是不屑的，而严莎却愿意遵循，认真执行，并且持之以恒，并使之成为习惯。

严莎就是这样以一种自然的心态面对工作中的一切，而她的实力和性格就在这种慢慢舒展的状态下，逐渐凸显。就像角落里的那簇雏菊，它只是努力的舒展腰身想要开出花来，远没有想到，其幽香已经在四周蔓延。

她开出了最美的本色。尽管后来深处高层，管理整个按对的她依然丝毫没有松懈。她说：“作为程序员最重要的品质就是责任和坚持，而责任并不只是一个程序员而是任何人都也应该有对自己说出的话和做过的事承担则责任的心。”

严莎把工作中看似不起眼的事当作最要紧的事来处理，这也练就了她“小事不怠慢，大事不慌乱”的优秀品质。为此，同事们佩服不已：“凡是难做、体量庞大的项目，她承担下来从没有怨言，并且都能给出非常漂亮的结果。”客户是她品质的直接受益者：“小严年轻有为，却有着中老年人才有的缜密思维，这是难能可贵的，我们的项目交给她绝对放心。”

一路走来，严莎就像一名无畏的展示，攻克了一项又一项软件开发，其成功案例在业界享有盛名。然而相比于工作业绩带来的成就感，严莎更享受工作过程中学习带给她的充实感。

“事实上做网站可以让你的视野变得开阔，可以让你了解各行各业，挖掘他们的一些专业知识。”严莎告诉记者，在做游游旅行网的时候，她领略到了许多旅游景点的旖旎风光，也学到很多旅游业的专业知识；在做江西律师网的时候，她了解到了律师行业的严谨和专业；以及一些商城应用、黄页和店铺等功能应用的网站制作和设计，严莎用好气的眼光打量着各行各业的新鲜事，兴奋不已。

“我们华夏互联已经启动Android的开发，并且走在江西软件产业的前端，这对江西的软件在移动互联网甚至更多的领域将带来更多的发展和机遇，我希望我是这一领域和机遇的领头者和创造者之一。”在讲述这些的时候，严莎一直微笑着。这种感觉缘自于她的性格，源自于她的阳光心态。她确实是一名微笑的战士，坚强勇敢，以无畏的心态政府一个个困难；乐观隐忍，以最真诚的微笑迎接每一个明天。

 

爱编程也爱生活

工作中的严莎态度严谨。想法独到，不编程的日子，投身生活，严莎却是一个爱生活、爱热闹的女人。问她怎样在这两个角色之间从容转换？答案还是：顺其自然就好。

她新欢选择阳光灿然的周末去郊外踏踏春，她喜欢和同事们一起在KTV纵情高歌到打烊，她喜欢在很多歌星期天慵懒地睡到自然醒，她也喜欢趁出差的空隙背着相机在向往中的城市走走停停。

走遍了天南地北，然而最让她情有独钟的还是南昌。你要问她为什么，她能理直气壮给你一连串答案：南昌的物价水平相对低，可以用有限的钱享受很多很多的美食；南昌是中国的水都，不需要忍受迎面风尘之苦，女孩子呆久了可以更加清秀；南昌的男孩子更温柔更贴心。。。。。。

说起家人，严莎流露出的亦是饱满的幸福。她很感谢亲爱的妈妈，“她让我们兄妹俩从小享受了优人一等的教育条件。而家庭带给严莎的影响，也是刻骨铭心的。”记得我上初中的时候，第一次到我爸妈工作的地方，见识到了他们工作的艰辛，也从中海知道任何行业都有其困难之处，只要我们坚持下去，就会走到下一个在站点的。”家人的坚强让乖巧懂事的小严莎从小就懂得了坚忍，这使得如今的她在工作中，一如既往地乐观面对。

叙说着那个生活的温馨，严莎的表情在慢慢舒展，透过职业套装，可以毫无阻碍地感觉到一个女人的幸福。这就是严莎，从女程序员到高级管理层，从IT精英到幸福女人，她可以做到如此自然。

往后的日子。严莎会仍然一丝不苟滴编写程序。她的双手会像魔术师的双手一样，把枯燥无味的代码变成更多丰富多彩的软件。

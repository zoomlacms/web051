      亭亭：开发部经理，专注于WEB前台研发、软件架构，具备良好的编程与系统框架设计能力，Zoomla!逐浪CMS5.x前台体验架构设计师。桃源才女恒心恒德，历经程序开发与互联网的风花雪月，更加知道专注的力量。勤奋而不守旧，自信而不骄傲，用技术改变世界，不做世界的随从，展示互联网的造梦师和建筑师风采。

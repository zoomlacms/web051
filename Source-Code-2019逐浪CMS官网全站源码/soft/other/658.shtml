快速部署-漂亮大方的2010节日祝福插件[适合逐浪CMS所有版本]

作者： 发布时间：2010-03-02 来源:本站原创 点击数：

详细介绍：

想在自己的网站上建立一个节日祝福？
想让更多的网友使用你的祝福转发？
想通过祝福获得更多的流量？
想让祝福有更多的个性？
现在，使用Zoomla!逐浪CMS，再加上本插件，就可以轻松的搞定这一切。
只要后台建节点、绑模板，一切全部自动生成，爽呆吧：）

1、skin直接覆盖根目录下Skin，不要管提示，选择“全部覆盖”即可。
2、将模板COPY到您的模板目录。
3、在Zoomla!逐浪CMS后台建立一个节点，命名为：2010祝福或任何你想要的名字
4、将“栏目首页模板”指向绑定[2010新春祝福.html]
5、将“栏目列表页模板”指向绑定[祝福生成结果页.html]
接一来你只要访问
http://你的网址/ColumnList.aspx?NodeID=[你的祝福节点ID]
就可以自由生成祝福、传递祝福了，痛快吧，嘿：）

在线体验：http://www.bsnkyy.com/ColumnList.aspx?NodeID=233&name=%e6%97%a0%e5%90%8d%e7%ab%b9

PS：本插件主要用的是逐浪CMS的数值传递敏捷开发，欢迎更多的朋友参加到我们的开发中来。
其中生成结果页模板的复制运作有两个，多放一个，让大家多一份选择：）

本程序适用于5.5前所有历史版本（含RC版、1.x、2.x、3.x、4.x、5.0、5.1等版本）升级到5.2版本，仅限于数据库存操作，升级完成后需要同步5.1版程序文件方可正常使用。
请严格遵从以下流程进行升级：
1、首先确认运行本程序的计算机是否安排了Microsoft .NET Framework 2.0脚本解释器（即.net2.0服务），如果没有请先安装本服务;
2、解压程序包后，双击[升级程序.exe]文件启动升级程序;
3、在程序对话框中配置数据库连接信息;
4、点击[开始升级]即可完成数据库升级；
5、升级完成后，请时将同一版本的程序文件更新到网站目录下，即可完成升级。

注意：
1、本程序支持所有历史版本升级到5.2程序;
2、解压程序包后，请不要删除、修改或改名同目录的文件和脚本，否则升级程式可能无法完成；
3、本程序支持您远程升级，即只要在本地启动升级程序，即可升级任何联网下的数据库；
4、对于大型数据库或服务器资源相对紧张情况下的升级操作，请保持耐心，升级完成系统会有相应提示，否则请不要进行退出操作。
5、任何时候在进行升级前，对原数据库和程序文件进行备份，都是一个好习惯，以免数据丢失。
6、升级完成后，更新源文件时请检查\Config\AppSettings.config文件是否为最新，并设Installed值应为true(详见http://bbs.zoomla.cn/)。
7、升级完成后，请进入后台检查各项配置，以保证正常使用。

感谢您的使用！

专业点评逐浪软件

我认为逐浪软件最大的优点在于强大的可扩展性和二次开发，与微软平台的无微结合，节省了我们大量的IT投入，这一点是难能可贵的。
--北京融合创意设计公司负责人张小姐

作为中国最大的律师区域门户，江西律师网在和逐浪软件多年的合作中，受益于其专业
-知名律师、南昌朗秋事务所合伙人刘平

我们公司的总部在美国，不可能所有的开发与IT需求与总部远程联络，正当苦恼于国内没有卓越的CMS能够兼容UTF8国际语言环境时，在开发者论坛上遇到了Zoomla!CMS，其良好的语言兼容性，且支持高端数据库访问，既满足了我们的需求，又让我们看到了国产软件的希望。
--美国globe tracker公司中国区外联部负责人李小姐

整合SNS，让我们节省了更多成本，同时让原有的数据库应用更加高效，领导和用户都说好用，这就是我们追求的！
--江西教研网负责人张先生

市场占有率超半的Oracle数据库平台，与逐浪CMS的良好结合，使我们的开发成本减少一半，更重要的是让我们的开发周期大大缩短，时间就是生产力，效率就是金钱！
--“博士”全国男科连锁医院IT主管张梁

逐浪软件与教育系统的完美结合，使学校的大量数据与OA办公成为可能，逐浪软件的优点在于其性能的扩展支持与功能强大，同时其缺点则是不够“商业化”，过于坚守开发人员的“技术贞操”，如同IBM电脑一样的专注技术……

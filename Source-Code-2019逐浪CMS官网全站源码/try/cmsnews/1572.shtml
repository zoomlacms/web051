搜索引擎已死-从团购到民营医疗与一淘

作者：本站编辑 发布时间：2011-10-22 来源:佚名 字体大小：【大 中 小】 点击数：

当青年团的领军人ljh先生进入美团，放弃大片的鲁赣市场时，我知道，新的时代来了。
这个时代的符号是：神马都是浮云。
圣人说，没有什么放不下的，放下就是轻装上阵。中国的搜索引擎已死。
 


一、淘宝在当街自淫
当淘宝声撕力竭的在中国一线城市投广告，
购物比价全在一淘
做开的妈妈，还得省钱有方
我是小明，我爱比价
真的啥都有？你信不信！反正
不比价，哪来的更好生活？
才几百家，这也叫团购大全？
购实惠，才够响当当，
比一比，才知道神不神奇
就这么定了？
……


附地铁广告原图（点击查看）：


　　
如同一个毫无剑术的草民，
挥着毫无章法的武器，
看似四面出击，
却毫无章法，
淘宝，也许真如马云hold哥所说的：打着望远镜都找不到竞争对手了！??


二、阿里系的阳萎和伟哥
　　从3721、到雅虎中国、到阿里搜索、再到一淘，无一不是展现出马云先生的失败与无能。
当地铁广告将所有的同行竞争对手都摆起来的时候，
所有的业界人士都知道，
并非如同一淘所声称达到“号令群雄”境界，
恰恰相反，
这些同行每一家都有独到之处，
而淘宝系生下一淘这个小孽种，
正在四处攀亲巴结，无一能超越。
 

三、搜索引擎的社会责任　

当一个声称可以取代所有社区的“一淘”出来时，
我们用它的搜索，任意商品，都会将淘宝商城和淘宝网放在第一位，
这样的平台，是不是可以取代所有的垂直商业门户？


马云和他的弟子们不会认识到这个问题。
因为他们已经忘却了社会责任，
他们只看到了百度的作恶境界和赚钱魔界。

搜索引擎的首要任务是提供公正的结果，
而不是自吹自擂的推广。

一淘显然不能取代所有的商业门户，
就如同淘宝显然不能取代其它垂直的商城
相信如同我本人一样反感淘宝这种小商品市场样式的平台人群不在少数。

这时，我们知道，淘宝已经在阳萎了
更大的杯具是：它还没找适合自己的伟哥。

三、社区营销的魅力
上周五和爱尔眼科（中国民营医院上市第一股）的华中区营销总监共进午餐，
对方问：如何提升爱尔的网络转化率？

我问：你认为爱尔互联网营销的最大的问题是什么？
他答：是没有足够的人来医院就诊。
我答：错，是没有足够好的公益形象。人们选择医院时，第一句话是问：XX医院怎么样，当别人说“不要去XX医院，是民营的”时，这家医院就彻底在准消费者大脑中被杀死了，永无再生可能。

我又说：　
很多企业为什么失败？
因为他们只想着当年怎么赚钱，却不想着以后怎么赚钱，
营销不是放眼未来，只顾眼前无异于饮鸠止渴。

我还说：
你看，每当夜幕降临时，一个个的社区连锁药店前，
成品药商打着“XX药业电影节”的横幅，一台电脑、一块幕布、一个投影，就可以放一场电影，给社区百姓带来多少欢乐。
事实上，这些看电影的人并非是真正的客户，但可以赢得金不换的口碑，
试问你们有实施过这样的营销吗？
你们愿意去做这样的公益推广来改进口碑吗？
----------------------------------------------------------------------------------------

今天的电商与搜索引擎行业也是如此，
人们并不缺乏所谓的“大”、“成功”、“有实力”的企业提供这些服务，
但缺乏真正公正的搜索引擎。

当公正不存在时，
360导航的流氓众生、门牙暴露、眼里不怀好相，一看就是三流胚子，也会被百姓大众认为是好人、侠士。
而那些戴着眼镜、穿着西装的所谓绅士，正是过街老鼠-人人喊打，
--因为它们根本赢不得人心。

当公正不存在时，
那些导航网址又迎来了新的一轮生成，
同时与社区SNS结合，人们宁可相信身边网友主页上的链接网址，
也不会相信搜索引擎排在前三页的效果，
--打死李彦宏他也不会百度搜索医院看病！！

当公正不存在时，
李彦宏不会百度找医院看病，
马云不会用一淘比价搜索--因为一淘根本无法给出任意一个商家的商品能比淘宝质优价廉的结果，
搜索引擎的社会责任何在？
 

四、另一个互联网

当JW拿着iphone和android的逐浪应用在莆田系老板面前捣腾时，
老板们总会说：如何转化成订单？

转化率已经成为了这个行业的最现实的词语。

什么才叫转化率？

当70%以上的广告费用都流为百度竞价，企业网络营销算不算成功？

当一个企业投入巨资开展所谓的“SEO”，最终连一个网站都是低端的“垃圾站”，算不算信息化建设？

当中国的搜索引擎不再肩负责任，
当中国最富进取的民营企业团体连数据挖掘与社区营销都可以淡定之时，
自建平台、
大数据库营销、
面向社区、
构建封闭的传播体系、
面向移动互联网的应用、
面向企业全员信息化与全员营销的生物链，
为我们描绘了新型电子商务突破与成长之道。


华夏互联与Zoomla!逐浪CMS团队已经做好准备，
带领企业走向新辉煌。

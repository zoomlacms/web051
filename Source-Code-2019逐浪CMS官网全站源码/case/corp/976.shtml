当前位置： 首页 > 案例中心 > 企业客户 > 四川环太实业有限责任公司
详细介绍:

世界苦荞在中国，中国苦荞在凉山，凉山苦荞看环太。

四川环太实业有限责任公司，一直致力于凉山绿色资源的开发和研究，成立于1992年，经过十多

年不懈地努力，一步一步发展壮大起来成为了四川省“省级扶贫龙头企业”。环太商标也获得“

四川省着名商标”“中国着名品牌”，成为凉山州首家通过的品牌；产品荣获“中国湖南首届国

际绿色食品畅销食品特等奖”、“中国绿色食品上海博览会畅销产品”、“西部博览会最受欢迎

展品奖”、“全国消费者放心产品”、“2010年最受女性喜爱的茶叶品牌”“中华国际养生节推

荐产品”等荣誉称号。环太系列产品进入上海世博会，并荣幸成为2010年全球妇女高峰会议馈赠

世界各国政要和商业精英的礼品。不俗的业绩使公司成为了凉山州民营企业中一颗冉冉升起的明

珠。

 环太企业充分利用凉山特有的原料，率先在国内研制开发出具有绿色有机原生态的“环太牌”

苦荞茶、还有具地方民族特色的“环太牌”凉山山珍系列产品和具有四川省着名商标品牌的各种

系列产品，这些产品已远销到全国各大、中城市，并在北京、成都、云南成立了分支机构。所到

之处，受到了各级政府和各方有识之士的真诚厚爱和大力支持。

近年来，环太企业生产的产品参加了在成都世纪城会展中心隆重举行的第十届中国西部国际博览

会和成都第十四届年货会年。凉山州委常委、西昌市委书记邓显祥也来到环太企业的展区给予了

高度评价。四川电视台新闻四频道、成都电视台第五频道都对该公司参加年货会做出了较大篇幅

的报道，也得到了更多消费者的喜爱和信赖。

茶是大自然给予人类最天然的保健良药，全球六大健康饮品之首。越来越多的人开始习惯于饮茶

。苦荞作为一个新兴的茶类产品，以香醇的口感和保健的功效受到了广大消费者的喜爱，进而掀

起了苦荞茶热，环太苦荞茶在2006年也被誉为了“四川最有特色旅游商品”之一。

来自大凉山最深处的原生态环境下的黑苦荞又名珍珠黑苦荞，有“黑珍珠”和“苦荞王”之美誉

，大凉山苦荞生长于海拔2200米以上的高寒无污染山区。其营养价值优于其它苦荞，不同产地的

苦荞所含微量元素的营养成分差别也很大，在 近代的研究中也发现，苦荞丰富的芦丁黄酮、蛋

白质、叶绿素、脂肪酸、碳水化合物、粗纤维、矿物质微量元素，能软化毛细血管、降血压、降

血糖，增强人体免疫力，有促进幼儿生长发育、预防冠心病发生的作用。而黑苦荞跟一般的苦荞

有一定的区别，富含生物类黄酮芦丁、粗脂肪、粗蛋白及人体必须的钙、磷、铁、锌、镁微量元

素比苦荞更高,环太所生产的黑苦荞茶采用凉山高寒山区所产天然珍稀的黑苦荞茎、叶、花、果

为原料，经严格挑选、超微粉碎后运用特殊工艺精心制作而成。生产过程中严格遵守安全、卫生

标准。采用全不锈钢机械生产，避免人为因素带来的污染，并在封闭无菌的条件下生产,其产品

更安全、卫生。生产出的黑苦荞茶既荞香爽口、汤色黄绿、清亮，耐冲泡，又保留了黑苦荞茎、

叶、花、果中特有的各种营养物质，科学做到营养、健康、味美、方便。所以黑苦荞相对来说有

较好调节身体的作用，长期饮用苦荞茶、黑苦荞茶有助于三高病人的康复，有助于体内毒素的排

除，达到增强免疫力、增强体质及养颜的功效，也是馈赠亲友的最佳时尚健康好礼！

环太公司的不断壮大，在全国也拥有很多的专卖店和经销商，而位于凉山西昌的“大凉山特产”

天禧店，是目前全国最大的一家旗舰店。在州内唯一获得AAAA级旅游购物类称号的商店。在这里

有凉山特有的各种旅游特色产品，有苦荞茶、苦荞系列食品、野生食用菌、菜等。

环太企业生产出的各种纯净无污染的绿色产品和黑苦荞系列产品的显着功效都迎合了现代人渴求

绿色健康食品消费的需求，细微贴心的服务也逐渐深入人心，被越来越多的消费者所喜爱。环太

企业将会为大家带来更多的健康和更多来自大凉山深处的纯天然绿色食品。

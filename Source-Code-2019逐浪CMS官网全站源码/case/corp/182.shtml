当前位置： 首页 > 案例中心 > 企业客户 > 盈高国际集团(中国) 有限公司
详细介绍:

盈高家具有限公司(www.gt-furniture.com)位于全国家具产业的中心，广东省。

　　我们从事经营门类齐全的优质家具产品，以优惠的价格，满足您的需要。我们乐于与客户建立长久的合作关系，从而协助客户的业务得到长足的发展。

　　我们的目标是通过优质的服务，迅速及时的报价，稳健安全的交付进度安排，为您提供做工精良的优质家具产品。相信基于盈高在生产制造和市场推广方面的丰富经验和成果，一定能够满足广大客户和消费者的要求！盈高通过对家具产品的设计、生产以及国内市场和海外市场的开拓。我们向市场供应优质的办公家具和民用家具，以富竞争力的定价获得市场的信赖。盈高通过严格的品质管理体系ISO标准，对生产过程进行全面和有效的控制。

　　盈高家具有限公司的合作伙伴和产品包括：
　　中联椅子厂：该椅子制造厂生产项目包括办公椅、酒店用椅，公众排椅、玻璃洽谈桌、百变会议桌等，主要的原材料为皮料，木材，金属以及塑胶，产品行销国内和国外，产品设计超前，生产供应及时，能够满足国际市场的不同需要。
 
 
　　中联办公家具厂：专业从事品种齐全的办公家具产品的生产制造，选材广泛，实木系列，胡桃木系列，泰国柚木系列，金属、塑料系列等。产品种类涵盖办公桌，电脑桌，办公区域组合家具，办公沙发，屏风等等，配合市场的需要，迎合客户的需求。

　　中联红木家具制造厂：我们从非洲和东南亚等地区采购优质和稀有的木材，制造出精美的仿古家具，红木家具，檀木家具以及酸枝家具等。部分产品全手工制作。本场出品的古典，仿古家具不但在广东省内广受欢迎，亦远销欧洲，美国，港澳台等国家和地区。
　　中联民用家具厂：拥有沙发、餐桌、卧室家具、酒柜等中高档民用家私制造实力，生产实力雄厚，拥有自有品牌，在市场上获得了较高的美誉，销售网络遍布全国省市和自治区，我们也具备丰富的OEM贴牌生产实力，与国际客户建立了牢固和长久的合作关系。
　　东莞联大家私城
盈高家具有限公司的长期合作伙伴东莞联大家私城，拥有四大商场，分布在东莞市区— 主山商场、石龙商场、常平商场、大朗商场，营业面积达三十万平方米。联大家具城为国内消费者以及海外买家提供了一个舒适便利的消费与展示场所。

　　盈高家具的高效经验丰富的专业团队，严格的品质监控贯穿在生产的每一个环节，涉及销售规划和原材料采购，制造以及运输过程等方方面面。

　　盈高在OEM/ODM合作方面富有经验，能够自行设计生产各类型的家具以满足客户的不同标准和要求。如果您正在寻求优质的家具供应商，我们盈高必定是您最佳的选择！

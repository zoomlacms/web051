发布时间：2010-07-21 分享网址

      乐平市为江西省计划单列市，1992年9月21日撤县设乐平市，由景德镇市代管，地处于赣东北，东接婺源县，德兴市，南接弋阳县，万年县，西接鄱阳县，北接浮梁县及景德镇市总面积约1973平方千米（城区约15平方千米），总人口约78.59万人，城区约16万人（2003年底），下辖14个镇，2个街道办事处，2个乡。

　　乐平乡地处临安市域中部南端，历来就有临安南大门的称号。面积43.7平方公里，距市政府所在地锦城街道59公里，距省会杭州112公里，北依大树华盖荫九州”的天目山，南紧邻桐庐县“一正一副”两中心的副中心集镇“中国圆珠笔之乡”分水镇。更位于“两江一湖”和杭州——黄山两条国家级黄金旅游线的最佳连接线16省道桐千公路的中段，随着杭州市域外环公路环境的逐步改善，杭徽高速的完工，临金高速公路的规划建设，桐千线改造建设，乐平下经桐庐“瑶琳仙境”，“垂云通天河”等溶洞景区，直达“千岛湖”，上经“天滩，石长城”，“大明山”等生态景区可到黄山，旅游区域优势逐渐凸显。

    乐平乡区域属天目溪河谷盆地，大多在海拔70米以下，地势低且平坦，部分区域矮山丘陵在海拔300米以上，天目溪，昌化江在乐平乡东乐村汇合后，自北向南呈狭长带状纵向延伸，水利条件较好，我乡属热带季风气候，四季分明，由于河谷盆地的原由，我乡常年积温要比附近地区高2度左右，适宜发展一些较低纬度的农产业，我乡山林面积48000亩，森林覆盖率85%，森林储积量6万立方米，与桐庐交界的白云山为最高坪，海拔711米。

　　乐平市位于江西省东北部，地理坐标为北纬28°42′14〞-29°23′24〞，东经116°53′36〞-117°32′40〞，北连景德镇，东邻德兴市和婺源县，南越万年县接鹰潭市，西毗鄱阳县。总面积1973平方千米，其中城区面积15平方千米。总人口80.3万人，其中非农业人口18万人。市人民政府驻洎阳街道洎阳南路。邮编：333300。代码：360281。区号：0798。拼音：Leping Shi。

　　乐平历史悠久、人文昌达。东汉灵帝光和元年（公元178年）始设县治，因“南临乐安河，北接平林”而得名，至今已有1800多年历史。1992年9月21日经国务院批准撤县设市，属于县级市，归瓷都景德镇市代管。全市共有国土面积1973平方公里，辖16个乡镇、2个街道办事处、1个农业科技示范园区和1个大型水库管理局；城区面积达15平方公里，人口15万人；全市总人口80万人。

　　千百年来，悠悠乐安水孕育了乐平灿烂的史前文化、优秀的传统文化、宝贵的革命文化和丰富多彩的民俗文化，尤以“洪公气节，马氏文章”着称于世。这里诞生了与《通志》、《通典》合称“三通”的《文献通考》这部流芳千古的巨大史书，以及毛泽东同志一生珍爱的《容斋随笔》这一垂范后世的历史佳作；源发了得到周恩来同志以“美、秀、娇、甜”高度评价的赣剧饶河戏；保留了400余座融建筑、雕塑、工艺、绘画、文学于一体被称为“中华一绝”的古戏台；培育了“一王二侯三驸马，四位左右一品相，五位状元、榜眼和探花，三百多位进士郎”；记载了方志敏、红十军等革命先辈留下的光辉足迹。

详细说明

经过长时间的潜心研发，调研广大客户的需求，倾心打造的Zoomla！逐浪CMS4.6版本即将同广大用户见面。在即将发布的4.6版本中，新增了全新的三维全景功能，让客户体验到三维全景震撼视觉效果！

 

　三维全景是基于全景图像的真实场景虚拟现实技术。全景是把相机360 度拍摄的一组或多组照片拼接成一个全景图像，也有通过一次拍摄就可以实现的，通过拼接或者一次性拍摄成像之后，经过一系列是处理生成全景图像，然后通过zoomla！逐浪CMS4.6版本中全新的三维全景功能实现全方位互动式观看的真实场景还原展示。

zoomla！逐浪CMS4.6版本中新增的三维全景功能我们将其放置在导航栏的扩展功能中。点击扩展功能之后，三维全景的管理界面就呈现在zoomla！逐浪CMS后台的左侧栏目中。
 

       三维全景功能简捷的管理界面，界面清爽且功能强劲

在观看三维全景的时候可以按照鼠标左键在画面上拖动任意观看，并能随时放大或缩小图片，并能全屏播放，让显示效果更加震撼。在展示画面的同时，可以通过添加背景音乐在欣赏全景画面的同时享受到高质量的音乐。

音乐播放器管理：可以方便的控制背景音乐

隐藏播放器：当不需要显示播放器时，可以隐藏，以免影响观看三维全景图片

 为方便用户更好的呈现三维全景效果，使呈现更具层次感，在zoomla！逐浪CMS4.6版本中的三维全景功能中提供一个简捷但实用的功能，通过一幅全景图中的进入、返回按钮可以直接进入下一个场景或者返回上一个场景，将多个三维全景进行融合形成三维全景虚拟漫游系统。

进入返回按钮：通过进入返回按钮可以进去或返回不同层面的三维全景图片

方便快捷的使用按钮：通过点击按钮达到更佳的显示效果，并有一键全屏功能

 

三维全景由于它的真实性、全视角等特点得到了越来越多网站人士的认同。zoomla！逐浪CMS4.6不仅融入了最前沿的三维全景展示技术，更是潜心研发，将自己的先进理念倾入其中，为的就是让zoomla！逐浪CMS时刻走在行业的前列！

在场景添加成功之后，为了方便客户能第一时间进行体验，我们增设了预览功能，同时也避免如果图片不理想时的盲目上传。在全景管理界面自动显示调用标签及图片的基本信息，方便进行下一步的操作！

 

可以说，zoomla！逐浪CMS4.6版本中的三维全景功能是网页显示效果一次重大革新，它改变了传统网络平淡的特点，让人们在网上能够进行全视角观察，而且通过交互操作，可以实现自由浏览，从而体验三维的 VR（虚拟现实）震撼视觉效果。

　　三维全景技术能得到长足的发展，根本原因是源自其卓越的应用前景和其带来的优秀的呈现效果。zoomla！逐浪CMS团队在潜心研发的过程中也在不断的吸收采纳广大用户的需求，由我们共同的努力让zoomla！逐浪CMS4.6版本中的三维全景功能在实践中体现出最优的实用价值。其主要应用领域如下：

　　●旅游景点展示

　　高清晰度全景三维展示景区的优美环境，给观众一个身临其景的体验，结合景区游览图，可以让观众自由穿梭于各景点之间，是旅游景区、旅游产品宣传推广的最佳创新手法。甚至可以用来制作风景区的介绍光盘，名片光盘，旅游纪念品等；

　　●酒店网上三维全景虚拟展示应用

　　在互联网订房已经普及的时代， 在网站上用全景展示酒店宾馆的各种餐饮和住宿设施， 是吸引顾客的好办法。利用网络，远程虚拟浏览宾馆的外型，大厅，客房，会议厅等各项服务场所，展现宾馆舒适的环境，给客户以实在感受，促进客户预定客房；

　　●房产三维全景虚拟展示应用

　　房产开发销售公司可以利用虚拟全景浏览技术，展示楼盘的外观，房屋的结构，布局，室内设计，置于网络终端，购房者在家中通过网络即可仔细查看房屋的各个方面，提高潜在客户购买欲望；

　　●公司企业展示宣传

　　●商业展示空间展示宣传

　　●娱乐休闲空间三维全景虚拟展示应用

　　美容会所、健身会所、咖啡、酒吧、餐饮等环境的展示，借助全新的虚拟展示推广手法，把环境优势清晰的传达给顾客，营造超越竞争对手的有利条件；

　　●汽车三维全景虚拟展示应用

　　汽车内景的高质量全景展示，展现汽车内饰，和局部细节。汽车外部的全景展示，可以从每个角度观看汽车外观，可以在网上构建不落幕的车展，可以制作多媒体光盘发放给客户，让更多的人实现轻松看车、买车。使汽车销售更轻松有效；

　　●博物馆、展览馆、剧院、特色场馆三维全景虚拟展示应用

　　●虚拟校园三维全景虚拟展示应用

　　●政府开发区环境展示

　　政府开发区投资环境，做成虚拟导览展示，可发布到网上或做成光盘。把开发区装到口袋带到世界各地变成了现实。向客商介绍变得一目了然，说服力强，可信度高。发布到网上，变成24小时不间断的在线展示窗口；

　　●装潢设计公司装修样板展示应用

　　装修公司面对客户，如何充分展示公司的实力和优势以赢得客户呢？通过三维实景照片，您便可以轻松的带客户参观装修样板房，把公司最经典的装修作品展现给客户。

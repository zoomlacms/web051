详细说明

        在面向互联网的仿真与实践应用领域，人们一直有一个远大的理想，就是把互联网与真实的世界相联，从而改进互联网在虚拟应用领域的作用，大大提升信息化的水平。

        因而这个领域，有两个方向最引人瞩目，一个是物联网，将传统行业与信息化连结在一起，目前已成为炙手可热之学术课题；一个则是面向互联网仿真的智能化应用，其中包括微软的可触摸屏幕、iPhone的多点触控以及一些更深领域的识别技术，在实践中逐渐走向成熟。

        Zoomla!逐浪CMS的在线试穿(戴)系统，是一个采用dotNET语言架构、结合FLEX与AJAX等多种技术结合的高端应用平台，也是逐浪软件上海研发中心成立后的首个高端产品。

       我们认识到，传统的B2B和B2C以及商业销售行为中，仅仅靠图片展示或FLASH等富媒体展示，远远不能满足用户的需求，比如一件衣服在线销售，用户需要知道这件衣服如何才能合身、穿在自己的身上感觉如何、针对自己的体型会有如何的变异等问题，而仅靠单向的一维展示，显然满足不了这样的需求。于是我们开发了这样的一套系统，力求使将虚拟的图形与真人的形象进行整合，从而开发出更加具备良好的交互的网站平台和新一代网站应用标准。

       首先我们在Zoomla!逐浪CMS中，建立了一个以试穿试戴为中心的商务应用中心，用户可以随时访问这个平台，并选择自己感兴趣的产品与模特，然后再点取商品进行试戴，如下图所示：

 

 

       用户在选择好了商品后，可以点取商品进行试穿(戴)，并可拖动大小，从而决定展示效果图的大小，建立起了一个实景的使用感觉。

如果用户对产品试用不满意，还可以重新选择商品或选择模特。

当然，仅仅靠单向的点选试用，依然无法满我们的应用需求，系统提供了关键的应用--在线真实试戴，用可以点选“摄像头”按钮，系统由此唤醒本地摄像头，进行拍照。

上图：启动摄像头前的友好提示。

上图：真实拍摄的场景

用户在拍摄了场景后，可以重复商品的选择、试穿、试戴，从而确定好购买意向，一旦确定，即可购买。

 

如上图所示，在整个场景中，图片的大小、商品的位置、试戴记录、隐藏眼睛等功能，都是可以自由操作，从而把最大的应用权利让客户选择。

 用户在选择好了商品，就可以进行在线购买，从而促成实际的销售业绩，商品购买操作台提供了丰富的商品信息，且完全采用无刷新的控件技术，从而为实现从应用体验到商品销售的完整过程，如下图所示：。

 

同时，我们还可以结合逐浪CMS的3D、视频等模块，进行更高层次的开发，未来这一模块将走向更高的的角度发展，致力于为高端用户提供领先的商业服务。

当然，这样的仿真系统也可以应用于网游、虚拟空间、社区等产品线，例如在网游中真人穿虚拟装备从而提升互动一体的游戏。

囿于商业项目开发的需求，在此我们仅批露这些信息，同时我们提供了一个在线测试体验的站点：http://a19.19.v.hx008.cn/try/ColumnList.aspx?NodeID=189&TryID=13
打开这个页面，选择试戴，注意：这是一个DEMO模型，只开放了第二件商品[测试板材镜架板材镜架]的试载，可以在线体验。

 

注：此模块为逐浪CMS商业模块、暂未整合发布于开放版本中，如有需要可以联系官方商业咨询。

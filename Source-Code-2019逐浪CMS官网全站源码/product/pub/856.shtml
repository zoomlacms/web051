详细说明

Zoomla!逐浪CMS是国内首家集成MSSQL与Oralce两大数据库高端CMS，摒弃低端模板嵌套业务，专注核心WEB计算与数据处理，为诸多软件厂商提供内核服务，开放式计算以宽广的胸怀长期在企业和高端门户系统开发领域乘风破浪、赢得了高端开发者的一致认同。

      

       在即将发布全新的Zoomla!逐浪CMS4.6版本，开发团队进行了前所未有的缜密细致的开发测试工作。完善并且新增了诸多强大功能，可谓是Zoomla!逐浪CMS在2010年的一个重大跨越。

在我们不断仔细体验全新功能的同时，今天让我们一同来体验下安装界面方面的一些新特点和改进。　

      整个安装界面采用蓝色的基调，新的蓝色色调给人一种轻松爽朗的感觉。进入安装界面后，一种不一样的清凉感受就出现在用户面前。全蓝色的界面，让人眼前一亮，营造了安装时的轻松氛围。

 

    在安装界面的下方新增了安装进度条，直观而形象的显示出当前安装的进度。　

第一步 阅读许可协议

建议在运行安装向导前仔细阅读程序包中的《安装说明》文档，如果已经阅读过，可以直接点击下一步继续安装过程。

第二步 进行运行环境检测,对数据库文件和常用文件夹及文件权限判断

新增的运行环境检测功能，整个过程分为四步检测，依次是版本检测、完整性检测、脚本文件检测、权限检测。

运行环境检测功能的目的在于安装前检测当前.NET的版本信息，是否能够支持Zoomla!逐浪CMS4.6版本；安装程序是否完整有效；环境的兼容性；并对相应权限进行检测，从而有效的避免无效安装！

第三步 进行数据库连接配置

此步骤中应该注意的是确保设置好的数据库中没有旧的数据表和存储过程，并且数据库口令不能为空，否则可能无法继续安装过程。

第四步 创建数据库表单与存储过程，显示数据库创建进度

Zoomla!逐浪CMS4.6版本提供数据库自动创建的功能，方便没有创建好数据库的用户。如果数据库已经创建好，可以选择跳过这一步，继续安装过程。

第五步 部署网站基本信息

部署完管理员名称和密码等信息之后就完成了Zoomla!逐浪CMS4.6版本的安装过程，接下来就可以体验Zoomla!逐浪CMS4.6的全新界面和强大功能了！

 

Zoomla!逐浪CMS团队不断的将先进的CMS理念融入到新版本中，同时也在不断的满足用户们的最新需求，所作的一切都是在不断的追求逐浪CMS的尽善尽美，都是为了给客户呈现最实用功能最强大的CMS！

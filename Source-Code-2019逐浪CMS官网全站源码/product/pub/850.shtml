详细说明

 

亲爱的Zoomla！逐浪CMS的客户们：

   您是否希望能够及时的了解以下信息？

        1.我的订单是否有最新的进展了？

        2.我的提问是否获得了新的答复？

        3.我与客服的交流有没有最新的回复？

 

在即将发布的Zoomla！逐浪CMS4.6版本里，我们充分的考虑到了客户所关心的，你想到的我们都做到了！这些你关注的信息将会第一时间以弹出气泡的方式提醒您，让你不再漏掉任何消息，让商机紧握手中。
 

此图为会员中心冒泡功能 提示当天订单数量

 

此图为最新有问必答的消息提示

 

气泡显示位置：屏幕的右下角。浅色的色调和会员中心的背景和谐呼应，让人有赏心悦目之感。

 

您可以通过点击气泡上的图标和提示信息链接，直接跳转到相应页面，方便及时查看。不再需要自行进入特定的位置，只需要轻轻一点。

气泡功能提醒的内容主要包括：订单管理、有问必答、在线客服系统等。一旦有最新的信息，系统将自动即时的弹出气泡提醒，让消息不再溜走！

 

订单管理、有问必答、客服系统是目前网站商务活动最重要的，也是实时性要求最高的功能板块，在瞬息万变的商海中，时间可以说是能左右结果的重要因素。让实时的动态即时的反应到网站上是成果的关键！可以毫不夸张的说实时气泡提醒功能必将成为商务活动的有力保障！

 

气泡功能看似只是一个个小小的消息提示窗口，但是它体现的却是Zoomla!逐浪CMS团队时刻为客户着想、不断从细节开发的理念，我们会不断的研发出更加人性化的功能更好的为广大客户朋友们提供优质服务。

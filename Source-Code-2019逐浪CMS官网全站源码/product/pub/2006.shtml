详细说明

 

Zoomla!逐浪智能平台(Super Plat)X1是由上海逐浪CMS开发的专业软件、硬件一体化平台，拥有国际上最优秀的软件与硬件设置。

该系统由上海逐一软件科技有限公司和江西逐浪软件科技有限公司联合研发，致力于利用最先进的WEB开发技术、硬件技术、控制技术和仿真应用技术，实现企业应用的一体化、实体化、持续化、立体化。

 

本产品一共分两个系列：
X1：基于普通摄像头技术进行控制，集成智能视频应用客户端。
X2：基于KINECT体感开发，具备良好的体感控制，并与智能触控等一体，具备更高加端的控件能力，能够识别手势、声音（此部份资料可参阅：http://www.zoomla.cn/corp/video/1839.shtml）。

 

 

Zoomla!逐浪智能平台(Super Plat)X1产品功能与特点包括：

  • 支持触屏、鼠标、键盘、视频等多方的输入，尤其是触屏输入，使操作更加方便。
  • 非传统瘦客户端模式，完全支持标准windows系统，既保证应用性能、扩展性，也节约了企业的边际成本。
  • 方便的展示客户产品。
  • 与传统客户管理（CRM）、进销存（ERP）等系统结合，无缝应用。
  • 高效的展示产品和门店。
  • 与Zoomla!逐浪CMS的三维全景结合一体，触屏启动。
  • 与Zoomla!逐浪CMS的3D展示系统结合一体，触屏就能启动。
  • 与Zoomla!逐浪CMS的在线试戴（试穿）应用结合一体，满足买家和访客的体验和数据采集。
  • 模拟展示、办公应用、监控管理一体，即可以使之成为展示互动系统，也可以接入办公应用，作为数据存储和小型网络中心；同时还可以进行24小时的实时监控管理。
  • 远程管理，远程开关机，远程监控，减少维护成本。
  • 与Zoomla!逐浪CMS的网站、商户、黄页、订单等高度结合、无缝化的管理应用，实现商家的卖场到网站的0距离一体化应用。

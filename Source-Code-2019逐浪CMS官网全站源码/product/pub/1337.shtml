详细说明

 当今中国，除了百度、阿里、腾讯外，还有谁是骑士般的风范？

那就是盛大！

作为游戏领域的霸主，盛大近年来的投作挺多，除了在游戏、电商领域外，在支付系统亦欲打造新的高度，盛付通支付平台的推出说明了陈天桥要在电子商务方面大干一场的决心！

作为国内领先的CMS厂商，逐浪CMS于2011年3月全面推出支持盛付通的版本（所有商业用户都可以通过客户中心获取并将全面开放免费下载）。

提到盛大，更多的人想到的就是盛大游戏，也许你和你身边的朋友也是盛大游戏中的一员。其实只要我们对盛大略微加深了解就知道起点、红袖、浩方平台、ET语音、酷6视频等等，很多网民熟知品牌都是盛大旗下的，而盛付通的发布必然会在B2C、C2C等电子商务领域掀起一个新的狂潮！

1.jpg(41.85 K)
2011-3-30 14:30:40



盛付通具有“支出和消费的场景合二为一”的特点，与其他支付公司相比，盛付通打造的是微支付平台，不仅金额小，更突出支付的便利性。微支付特点是“全、易、强”，所谓全，指全国有35万经销网点；易，指支付便利，十秒到账率高达到99.99%；强，指每分钟近6000笔的交易频率，一年交易达31亿笔，并拥有强大的风险控制体系。

盛付通更开创了用户无需开通网银，只需要用户有一张银行卡、一部手机便可以简单快捷地完成支付，一键傻瓜式支付让普通百姓也能学会电子支付，走进电子商务。

可是你的商城竟然不支持盛付通，甚至你对盛付通的了解远不如你的客户，那么你丢失的可能不仅仅是客户！

然而面对盛付通的接入问题你又无从下手，怎么办？

使用我们逐浪CMS商城系统的客户是完全不用担心的，逐浪CMS在第一时间就加入了盛付通的接口。

如图，在逐浪CMS商城系统的后台→支付平台→增加支付平台→接口类型选择

2.jpg(55.35 K)
2011-3-30 14:30:40



点击右边的下拉框就会看到除去我们网友常用的快钱、支付宝、财付通、银联接口，还有最新加入的盛付通接口。

3.jpg(37.14 K)
2011-3-30 14:30:40



选择盛付通接口，在添入对应的支付名称、商户ID、安全验证码，选择商家分成点击保存就成功了！

同时还可以增加多个商户ID供客户选择，自由设置盛付通支付是否默认、移动位置、禁用和删除功能。

组卷系统基本使用示意与功能展示

关于本功能的简介:
展开简介
详细说明

组卷系统的原理是基于站点后台题库，用户前台可以选择题目，并通过服务器端Office组件生成符合规模的试卷，从而减轻教师的工作负担，也为科学分配试卷提供便利。

因而部署组卷系统的首要条件是，服务器商必须安装微软公司Office套件中的Word组件，这是必须条件。

在初次安装好逐浪CMS之前，即自带组卷系统，访问网址是：http://网址/Questions/Default.aspx?id=1

点击进入后的前台界面：

用户在可以在前台页面选择试题，加入试题蓝，然后根据自己的需要生成符合规格的试卷，生成目标格式为WORD2003版的DOC格式。

下图：试题选择界面

下图：选择好试题后可以进行上下迁移调整试题在卷子中的位置

下图：系统提供8开和A4两种幅面的组卷

后台教育系统-组卷系统中，可以进行管理，其中菜单包括：系统试卷管理、试题管理、分类和宣、用户试卷管理、用户试题管理。

试题分填空、应用两种题型，并可以设置难度、复杂、分类、所属试圈等，其编辑界面如下：

上面的聊天记录，是某同行CMS厂商针对逐浪CMS的一段攻击。
我们来过滤这一段聊天记录的关键要点：
1、刻意针对逐浪开发团队的一个下载路径编辑误差发起攻击（随后即已修正），以此恶意诽谤我方为“骗子”、“坏蛋”；
2、这家CMS厂商可能最卓越的技术在“模板设计”，也就是界面开发，并擅长于专业的开发，姑且不论这种借奥运会宣传和盗用www.bj2008.cn官方网站设计版权的行为是否合法，但他们已经认为自己是“原创”了奥运会模板。
3、自封为“专业模板设计”的CMS厂商，是否与其开发的身份相合适，这点姑且不论，但想请问的是：你们的奥运模板就是原创了吗？学术间的借鉴难道不合法吗？学习难道可耻吗？逐浪CMS团队不会简单的抄袭和拷贝，我们只会面向客户和技术从事开发。
4、我们认为任何的软件与开发，互相借鉴学习、互相交流沟通、甚至是源码与技术交换是很正常的，只有不断的交流开发，才能有最优秀的程序，才能为客户提供最优秀的产品—这也是真正的开源精神之所在。
5、我们感谢所有鞭策我们的人们，即使是诽谤、即使是攻击、即使包怀恶意，我们也能够面对，至少我们认为我们做得不够好。
6、我们的逐浪CMS从来不敢自封是国内“CMS之母”，我们欢迎所有的开发者来学习、申请源码，我们认为我们的CMS是“最差”的、所有的CMS都比我们强大—只有把自己放到很低的位置、把眼光放到最远的未来、把学习的对象放到国外同行中去，才能铸就真正的世界级网站内核管理系统。
7、产品的好坏，市场和媒体能够说话，CSDN、网易、新浪、凤凰网、驱动之家不是瞎子，包括军区在内的客户不是瞎子，事实告诉了我们：逐浪CMS是经历得起考验的CMS！诽谤攻击我们的朋友们，如果您认为客户和媒体是瞎子，那请继续您的愚蠢想法吧！
 
再次感谢所有的朋友，不管立意为何，都将激励逐浪CMS的自主、奋进和创新之路！

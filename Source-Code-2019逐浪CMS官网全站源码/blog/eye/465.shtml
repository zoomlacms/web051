从“人间正道”领悟的八条软件开发与项目管理心得

作者：发哥 发布时间：2009-07-22 来源:本站原创 点击数： 分享网址

博主：发哥
个人座右铭：建网站，选逐浪。 易上手，功能棒。 质量好，服务善。 高性能，最划算。

 

电视剧《人间正道是仓桑》是以发生在湖南醴陵的杨家为故事基点。
事实上，湖南醴陵并没有这样的一个“杨家”，但是这里孕育过XX党第一书记李立三等在内的精英，在建国60周年的巡礼中播出此片，笔者认为显然包括了对李立三等先驱的褒赏。
从湖南醴陵向南三十公里，是一个叫“茶陵”的县城，那里是中国工农红军第一个革命根据地所在，现北京军事博物馆关于红军的第一段历史即从茶陵根据地开始，其后方是井冈山英豪聚义的故事开始。
常年的奔波各地，作为中国程序员的一份子，自然对我党、我军的创业史留意收集，因为我们的事业走的也是一条白手起家的创业之路。而我的太太刘璐莹小姐出生于茶陵望族，更让我对其中感受颇深。
在此，收撷几条鄙人从《人间正道》电视剧中的心得，与大家分享：
一、         程序员是需要理想的，虽然社会提供了许多所谓的“发财捷径”，但是没有理想者最终还是会走向失败。
影片：从瞿恩、瞿霞、林、立青等人的发展轨道中，我们可以看到一个个鲜活、为了理想而奋斗的人生，正是他们的理想，照亮了自己、照亮了社会、照亮了未来。
二、        即使你拥有满腔的热情、忠诚和才智，跟对人、站对队往往会决定整个项目的结局。
剧中主人公杨立仁，从黄埔军校教官、到抗日精干特工、再到反共一等功臣，拥有聪明才智、更具备对国民党“蒋公”的无限忠诚，可最终的失败苦楚，不知有多少精英会有一同感慨。
不幸的是，现实中，不少年轻的程序员，都是以“学习经验”为目标，基于是“先学一下，等掌握了就换个山头谋新发展”，“要签三年合同是不可能的，因为未来三年对我来说太重要了”，这里，要请这些年轻的所谓程序员们（笔者认为他们事实不算合格的程序员）和国民党反动派“杨立仁”对比下，“民工”思想下的人们，才志均不如人，何谈成功？
三、        人生往往只有几个重要的转折点，而为这个转折点所做的准备，远远超过事件的本身。
想我XX党带领全国人民的解放战争，通过三场大战，即初定江山。
不少人总认为当年的战争全凭的是人民解放军的巧打、游击致胜，可当你战在苏北平原一望无际的广亵大地上，你就知道这是多么激烈的战争了。一望平川、毫无掩伏，凭的是真正实力！
中华人民共和国的创立是真正能力和勇者开山，而不是投机取巧的“敏捷”之道。
四、        不要以为只有写“核心源码”才是编程，没有细节和基础技术，连“编程”的门都不能算进。
逐浪CMS软件团队的招聘中，经常会遇到一些心浮气躁之辈，他们认为只有使用Microsoft Visual Studio工具才叫开发，可连一个基于的CSS属性都不会，真不知这样的人才在WEB开发大行其道的今天，应如何生存下去。
剧中主人公杨立青同志，战士、教官、特工、后勤、排长、连长、学生、司令，每一个职位都是以团队的需求为出发点的，只有一步一个脚印，才能走出精彩的人生。
五、        你可以放弃选择，但不能选择放弃。
这句话我最早是从上海篱笆网的营销总监刘继汉先生口中学到，但看看电视剧中的女主角，一个个的献身于革命事业，即使是与国民党的交往中，利益相诱，也能把握住自己的立场，不得不让人钦佩。
现实中，不少程序员被“收买”，从乙方转身跳到甲方，再从甲方程序员转身变为“只说不做的小混混”，这样的事件屡出不鲜。当事人认为“有机会就不能放弃”，事实上你何曾是俘获机会，而是机会俘获了你！
六、         最好的工具往往不会生产出最好的产品，最恶劣的环境往往诞生最惊心动魄的篇章。
想想国民党的“美式机械师”，靠着从美国“原装进口”，在人民解放军的自力更生、自主生产的战争下，毫无生机可言。
年轻的程序员们总是喜欢崇拜建模、UML、架构、领导、需求、团队管理之类的词汇，而忽略真正的开发与测试能力。要知道，再丰富的知识，没有行动去践行，也是没用的书本文化。
所以我们的观点是：战场上见输赢，有能力就做出程序来，否则一切空谈都是反动派。
七、        现场，只有现场才是程序员的天坛。
天坛，古代封建皇帝封爵加后的地方，代表了国家的最高神旨。
现场，程序开发现场，才是程序员真正的天坛。
国民党的精干美制军队的指挥思想是：司令部离前线只有100米、师团旅离前线只有50米，营连排只有冲锋陷阵，这样的军队才有战斗力。
XX党战无不败的宣传理念是：在国民党军队中间发展革命的力量，无论何时何刻都注意白区群众的思想宣传工作。
蒋家天下失败的标志是：长江以北战败，即将大批珠宝财富转运台湾，远而避之。
如果说，整个团队在加班，而另一个在嘲讽，那这个嘲讽的人再才富五车也是一个无益于团队的人，他的命运只有消失和淘汰。所以逐浪CMS开发四大原则中一点是：我在现场，因为平等和责任让我不会放弃。
八、        勿以利小而不赢，勿以利大而倾巢。
积小胜成大胜，用心经营，才有我军我党的成功。
做生意、接订单、做开发，也同样，不以利益的大小为评价，无论是百万千万的订单，还是百元千元的订单，都应倾力做好。
“三年不开张、开张吃三年”这样的心态，是走不出卓越的软件公司的。
所以笔者常会说：“软件公司是怎么死的？都是自己做死的：第一年做产品赢得认可、第二年做政务系统和大象跳舞，第三年玩概念销声匿迹。”
试问一下朋友们，当IBM、华为、金蝶软件、东软这样的公司都在世界各地奔跑为一分利而追逐时，我们的民营企业何尝不是“只做政府‘大’单”、“出差异地订单太累一般不做”的生存着？
国民党从抗日战争到解放战争，无一不是恃势自傲，完全以赌徒心态去经营战场，以致于若干江山在三场歼灭战后居然是无兵可派、无将可陈，其败亦必然（如果国民党也能掌握我人民解放军的游击、迂回战述，则大可避免全迁消灭、战事急剧变化之后果）。
 
欲行正道，必历仓桑。
笔者不是每天都是准时下班的人群，也不是整天都坐在电脑前看电视剧的无聊之辈。
偶然的看到《人间正道》电视剧，即让我对其爱不释手，故在差旅途中整理出这段文字与广大同行共勉。
而逐浪CMS的同事们，只要不是陪太子读书之辈，相信假以时日、脚比路长、历经艰辛，必将成就大道—因为我们胸怀理想、拥抱真知、日行日践、团结奋进。
 
附：湖南茶陵在明代曾诞生过文学之花李东阳，“茶陵诗派”在历史上享有盛誉，有机会笔者将抽空就此作文以飨读者。

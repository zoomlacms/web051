从WM与SB的差异化竞争谈WEB设计及网站功能化开发思路、以及未来CMS产业竞争

作者：yansha 发布时间：2010-09-14 来源:本站原创 点击数： 分享网址

 有一个城堡主要举行重要宴会,城堡里的每一个人都在为这个宴会作准备,一切都准备就,忽然发现宴席上没有准备鱼，也就在这个时候，天下起了倾盆暴雨。、
说来也巧，风雨之中，一个贫穷的渔民来到城堡，他背着一条非常大的鱼，堡主很高兴问他“这条鱼多少钱？我把它买下了！”
渔民说出来的话，让在场的每一个人都感到吃惊：“鱼的价钱就是用鞭打我100下”。
“真是胡说八道。”堡主断定渔民是在戏弄他，恼怒道：“你是开玩笑吧，快点告诉我，买你的鱼需要多少钱？”
渔民还说回答说要鞭打100下。
堡主拿渔民没办法，只好让手下按起他鞭打，1、2、3、……
打到10鞭时，渔民喊起来了：“住手，我有一个伙伴，他也入了我的股份。”
“什么？”堡主喊道“世界上有你一个这样的疯子也就够了，竟然还会有两个？你的伙伴在哪里？快把他带到这里来！”
渔民说：“我的伙伴不是别人，正是你们的看门人，他不让我进来，除非我答应给他9成的鱼钱，所以，我必须兑现我的诺言”。
堡主听后哈哈大笑：“把守卫带过来”。
守卫带来后，堡主一声领下，一五一十的鞭打起来，一连打了90下，才把倒霉的看门守卫放了。
贵族随后送给了渔民一份厚礼，以表彰其智慧。
--今天，我们不断的强调行业标准和筑城围攻，是不是能从中看到更加开放的视野？
 
 
 
引言：Windows Mobile（以下简称WM）和symbian(以下简称SB)是当今世界的两大流行手机操作系统，二者在智能手机操作用户端市场竞争近乎白热化，而各呈现出不同的发展趋势，本文试从多个角度分析差异，并引入到WEB设计、网站开发以及软件研发领域，作有益的探讨研究。
 
UI界面之启发
无可非议，在OS系统的界面设计上，WM是走在前面的，继续了Windows设计的精华，WM上依然提供了多任务、统一开始菜单、标准化PC虚拟空间划分等思路。
而SB在UI的设计上，不仅要避开WM的固有思路、从而消除专利的限制，更在于创造一种全新的习惯、对手机移动OS系统进行全新的反思。毕竟基于手机的系统，是与PC上完全不同的使用习惯，与传统电脑是有区别的。
我们首先来看两种不同介质上的带来的区别：
传统PC是有大屏幕的，而手机OS是小屏幕的。如果将传统PC的UI按同比的缩小或精简，显然不利于操作使用。这方面，WM的UI显然有这方面的问题，比如开始菜单过小，点选时如果不使用触摸笔有时用户常常会反馈操作困难。
这一点，SB系统的设计显然是经过深思熟虑的，甚至在97中唯一的功能键就是启动“程序”命令，显然方便—当然，这还得益于手机厂商的紧密配合，而SB就是这样的一个操作系统，即使他的厂商有时只有诺基亚一家，但它已经打通了与用户交流的七经六脉！
传统PC是面向功能扩展，而手机客户端的目标可能是固有的功能，比如通话、上网、IM、邮件。这方面，SB要做得更好，因为它知道手机客户端知道自己的目标是移动电话、上网和邮件事务处理，而WM的表现是迷失了自我，甚至梦想着有一天手机能变成PC的取代品，这种迷失自我是至命脉的，这也就是后来Iphone异军突起的一个关键—只要在音乐和图片拍摄浏览上提供足够“炫”的体验，就能让用户拥抱。
 
专利化的回避
殊途同归的追求
准确定位的重要性
有舍有得的发展战略
用户体验的感观需求
 
 
回到逐浪CMS的研发课题上，逐浪CMS不同于其它的厂商的CMS在于：
首先，逐浪CMS不是一家单纯的“为了CMS而做CMS”的厂商，每一个开发成果，都会紧密的应用到网站上开发上，因为其母体华夏互联hx008.cn本身就是一个专业网站开发工坊，就如同SB系统的母体NOKIA，本身就是手机厂商。
理论派的厂商，无论在敏捷上，还是扩展功能上，都无法达到真正的网站应用需求。
其次，基于对用户群体广大的理解，曾经WM也是智能手机的“标准配置”，多少达人以拥有一部安装了WM的MOTO手机为豪，而今天又有多少人关注？
从中看可以看看出，所谓的庞大的用户群体，事实上在真正的商业应用前，是多么的不堪一击。应该说，我们的逐浪CMS新BI应用，看到了未来的云存储应用，跳出当前CMS厂商过于强调“模板嵌套”、“安装就用”、“千站一色”的局面，以更高的商业智慧来面对未来。
 
未来的网站是什么样的呢？
应该有更多的网站存在。
应该有更多的应用。
 
就好比当年的广告设计一样，
要用很多复杂的工序，
才能在外墙上设计出大型海报，
而后来的喷绘技术与喷绘设备，
彻底取代了传统广告的手绘，
也不再需要院校学生们埋怨才无用武之地了。
 
当年的网站设计，
也是需要复杂的过程才能，
编码、UI、数据库、脚本、CSS、平面……
一切显得非常复杂，
而CMS结束了这种无序的时代，
为网站开发带来了福音。
 
那么未来这些喷绘机们如何竞争呢？
要看厂商能否提供更多的权益，
同时要看厂商的设备是否与广告制造商紧密结合，
最后还要看到，
这些喷绘机卖给的广告制造商们，
始终是行业最低端的商家！
高端的商家是什么？
高端商家们出卖智慧和标准—比如4A。
 
为用户提供商业智慧，
为客户提供更加易用的标准工具，
逐浪已经驶上了这辆快车，
而华夏互联的母舰即将泊岸！
 
 
注：亲爱的同事，这是一个有趣的话题，写这篇文章的最初目的，是为了分析WM和SB两个系统间的差异。
不得不承认我们的好高骛远，这两个系统间的研究资讯汗充牛栋，可是真正的研究，对于一个WEB开发者和CMS研发者来说，最缺少的是什么？
最缺少的是对产业的反思、只会顾及目前利益，当产业危机到来时，尤不会反思，最终走向消亡。
于是文章的后半部份转而写向了对整个产业的研究，当然，其中涉及未来平台的反思，很多思路并不成熟，又或已是公司实施的一部份，希望大家能仁者见仁、智能见智，给予指导批评。

走进西南-逐浪软件与客户手牵手[西南行记之一]

作者：发哥 发布时间：2010-09-14 来源:本站原创 点击数： 分享网址

博主：发哥
个人座右铭：建网站，选逐浪。 易上手，功能棒。 质量好，服务善。 高性能，最划算。

 

脚比路更长，只要有客户的地方，就有Zoomla!逐浪CMS的亲密服务！
当国航的班机徐徐停在成都双流国际机场的时候，我们迎来的是客户的亲密微笑与热情招待。那一刻，凌晨四点就起床的疲惫、五一无休的倦怠、远在他乡的忧虑都不再存在。
时间是2010年5月1日早晨九点，为期限三天的Zoomla!逐浪软件西南客户服务之旅已经开启！
 
关键词：Zoomla!逐浪CMS来自源于用户，必将紧密服务于用户
客户服务的第一站，是来离成都近二百公里的广汉现城网。这个出土过三星堆、古称雒城的文化古城，在互联网的今天正散发着智慧的光芒。
现城网的李、谭二人并肩合作，致力于打造当地最具价值的信息门户。客户是购买了我们的政府版系统功能，根据服务规范，我们提供基于线上的培训与服务支持，并无上门服务这一要约，但为了满足客户应用、并能为客户提供更多的紧密支持，上海逐浪软件公司售前顾问主动提出上门服务，由我们提供全程的技术支持。
在广汉现城网，我们提供了从UI到架构、从布署到基础应用、从深层开发到敏捷应用等的完整细致的讲学与培训，细致与耐心赢得了客户高度认同。

关键词：Zoomla!逐浪CMS仅仅是一张白纸
精至服务是客户认同的基础，不少用户在构建站点时都会遇到字段不能扩展、表单不符合要求、节点与功能表现形式无法满足实际运营需求等问题。
Zoomla!逐浪CMS其中的内容管理、商城、黄页、网店、模型、标签以及独立的模板引擎、再加上SNS、WAP整合其中等功能，开发者只要进行良好的构架和细致的开发，就能建立起功能齐全、应用便利的网站。
在客服现场，甲方工程师的一句：“通过这些天的使用，我们感受到逐浪CMS的功能实在太强大了，Zoomla!逐浪CMS仅仅是一张白纸，我们可以从中自由的开发！”让我们身心感动。
虽然国内CMS市场高低不齐，但是逐浪CMS从一开始的定位就抛弃了“套模板”的简单发展思路，而是以功能为优先，虽然在低端用户市场推广因此受阻，但也让我们赢得了更多的开发者们认同。未来的发展，将不断的追求功能与效率，为广大开发者谋福音！

关键词：五易其稿不厌
做旅游的高小姐是逐浪CMS早年的一个客户，通过逐浪CMS的应用，使其对电子商务、互联网、网络营销有了更深的认识，走上了站长之路，用她的话说是：不再是“开着网店亏本进货”。
于是我们在那个热情洋溢的晚上，一群人坐在“麻辣空间”火锅城，上到百度Google、下到传统行业电子商务如何应变发展，天南地北的侃起来了。
高小姐最早的需求仅仅是为了建立一个旅行社网站，后来通过自己的学习、逐浪软件售前顾问的指导，逐渐认识到，网站建立的意见不仅仅是做百度竞价广告这么简单，深层的意义在于吸引客户认同品牌、整合线上线下业务资源，构建起信息化的商业体系，于是逐步校正了自己的站点思路--虽然高女士的校正使逐浪软件团队的开发难度和精力投入大大增多，但我们从没有一丝怨言，我们乐于看到客户的成功，我们更有赖于客户的成功，方能铸就逐浪CMS的成功，一步一步的支持客户发展，方能得到客户的认同。
高小姐只是逐浪CMS诸多用户中的普通一员，地不分南北、客户不分大小，未来我们依然将扎实基础、服务每一个创业中的朋友。
 
关键词：我们不仅要的是软件，而是理念与管理思想
CMS系统是什么？
字面的意思仅仅是内容管理系统。
而逐浪CMS不仅仅是定位于提供内容管理这么简单，从早期的项目管理，到4.5版本崭露头角的BI应用，集成客户管理、项目管理、组卷、3D系统等应用，逐浪CMS的定位是企业管理的云通道，更是企业信息化的智能应用中心。
从广兴到彭山、从彭山到成都，客户朋友们一路和我们交流的都是如何围绕着互联网赢利、提升传统产业的竞争力。我们也知无不答、倾心聆听，并为客户提供完整管理咨询与案例诊断服务，一步一步的为客户诊断，导入全案策划。与客户肩并肩奋斗，共同赢取光辉未来。
 

脚比路更长，
如同我们的企业格言，我们始终恪守为客户提供一流的服务。
三天两夜，除去睡觉，连吃都是不断的就项目细节、软件开发、网络营销进行沟通交流。
当我们冒着雨赶到双流机场，连夜赶回上海时，我们知道我们将一夜奔波无休，虽然错过一场世博假期，但我们赢得的是更精彩的客户服务西南行！

附：热火朝天的火锅城和赤膊上阵的场面

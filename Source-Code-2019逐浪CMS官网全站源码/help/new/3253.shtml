逐浪CMS论坛社区功能管理菜单在哪里？以及装载帖子数据表是哪张？

作者：本站编辑 发布时间：2018-02-27 来源:佚名 点击数： 分享网址

问：逐浪CMS论坛社区功能管理菜单在哪里？以及数据表是哪张？


答：进入后台，点击内容-百科问答帖吧，就能管理社区论坛。

其中点击版面图标，进入帖子管理。


数据表为：

SELECT TOP (1000) [ID]

      ,[Gid]

      ,[CateID]

      ,[Pid]

      ,[ReplyID]

      ,[ReplyUserID]

      ,[Title]

      ,[MsgContent]

      ,[IP]

      ,[ColledIDS]

      ,[MsgType]

      ,[Status]

      ,[CUser]

      ,[CUName]

      ,[CDate]

      ,[Subtitle]

      ,[PostFlag]

      ,[Style]

      ,[OrderFlag]

      ,[HitCount]

      ,[IDCode]

      ,[LikeIDS]

      ,[R_CUser]

      ,[R_CUName]

      ,[R_CDate]

  FROM [f_ziti163_com].[dbo].[ZL_Guest_Bar]

其中：

  [OrderFlag]=2   全局置顶
  [OrderFlag]=1   版面置顶

为何本地有时打不开逐浪CMS-附Chrome\Firfox无法访问的web站点的端口列表

作者：本站编辑 发布时间：2014-10-05 来源:佚名 点击数： 分享网址

有朋友在开发逐浪CMS过程中，会遇到在本地建立站点，在IE下能打开，但在chrome、火狐等浏览器下打不开。
原来chrome、火狐浏览器为了避免一些端口与常用软件冲突，预先进行了规避，所以打不开。
所以建网站在本地IIS部署时，建议避开这些端口。

PS:又有网友说，我就想用chrome访问这些端口，有没有解决办法呢？
目前的解决办法是： 在你的Chorme 快捷方式中加入这个字符串--explicitly-allowed-ports=xxx xxxx是你的端口号 

例如：你要开启8888 端口号， 

则你的快捷方式中的内容可能是 C:\Documents and Settings\User\Local Settings\Application Data\Google\Chrome\Application\chrome.exe --explicitly-allowed-ports=8888

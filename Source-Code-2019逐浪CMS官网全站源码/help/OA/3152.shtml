如何清空逐浪CMS之OA办公套件中的文件记录

作者： 发布时间：2017-04-14 来源:佚名 点击数： 分享网址

问：如何把Zoomla!逐浪OA系统之前收文和归档公文里面的所有文件都删除掉？


答：可以。


DELETE FROM ZL_Mis_APPProg  --步骤会签信息
DELETE FROM ZL_OA_FreePro   --自由流程步骤信息
DELETE FROM ZL_OA_Document  --公文记录
DELETE FROM ZL_OAC_         --扩展字段记录(如存储期限等)

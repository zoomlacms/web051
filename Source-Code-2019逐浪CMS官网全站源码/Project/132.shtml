逐浪CMS解决方案的优势

 

1967年，哈佛大学的心理学教授Stanley Milgram(1934～1984)创立了六度分割理论，简单地说：&ldquo;你和任何一个陌生人之间所间隔的人不会超过六个，也就是说，最多通过六个人你就能够认识任何一个陌生人。&rdquo;按照六度分隔理论，每个个体的社交圈都不断放大，最后成为一个大型网络。这是社会性网络(Social Networking)的早期理解。后来有人根据这种理论，创立了面向社会性网络的互联网服务，通过&ldquo;熟人的熟人&rdquo;来进行网络社交拓展，比如ArtComb,Friendster，Wallop,adoreme 等。

但&ldquo;熟人的熟人&rdquo;，只是社交拓展的一种方式，而并非社交拓展的全部。因此，现在一般所谓的SNS，则其含义已经远不止&ldquo;熟人的熟人&rdquo;这个层面。比如根据相同话题进行凝聚（如贴吧）、根据学习经历进行凝聚（如Facebook）、根据周末出游的相同地点进行凝聚等，都被纳入&ldquo;SNS&rdquo;的范畴。

 

最早的交友网站纯粹用于网上寻找配偶等应用，随着互联网应用的不断深刻，交友的概念已经不断的深化，甚至于成为考核网站访问体验、互动性、能否粘住用户、提高用户活活率的一个重要参数--传统的网站信息数据庞大，却不能提供精准的分析，也不能粘合用户，从而造成&ldquo;数据陷阱&rdquo;。

专业解决方案、满足经营需求：Zoomla!逐浪CMS的开发团队，拥有丰富的经验，早在ASP时代，就具备了专业的交友网站的开发，一大批婚介门户即由我们开发，脱颖而出，现在依然活跃在商界版图上。

而全新打造的Zoomla!逐浪CMS交友网站解决方案，则采用最新的微软公司.net2.0技术构建，从技术上更上一层楼，值得用户信赖。在应用上，我们有丰富的数据库挖掘分析经验，甚至于为用户提供了基于C/S线下的软件，配合网站经营使用，满足性能应用。

人性模块设计、贴近访客使用：Zoomla!逐浪CMS交友网站解决方案同时提供完善的用户表现层设计，完全符合W3C标准的界面设计，专业为用户提供的开放插件开功能，用户点评系统，留言系统，贴近用户使用。

已取得成就

 

在这个模型中，我们把受众接触信息分为寻找信息、筛选信息、习惯匹配三个阶段。

在&ldquo;寻找信息&rdquo;阶段，信息是强势，而受众是弱势，这时谁掌握了最新的工具（上网），谁就占据了优势，这种时代已经在日进千里的互联网进程中成为了历史。

 &ldquo;筛选信息&rdquo;阶段有可能就是我们现在所处的阶段，网站异常繁荣，却良萎不齐，缺少权威的工具和标准来衡量，这时需要网民自己去筛选，筛选的结果自然是一些网站淘汰，同时网民的习惯、胃口也变得逾加刁难，&ldquo;君择臣、臣亦择君&rdquo;用来形象这个时代的网民和网站关系最最为贴切的。

 &ldquo;习惯匹配&rdquo;阶段则是网站充发展后的结果，在这一阶段中，网民有自己的价值观、上网习惯和社群理念，他们会根据既定的程式来接收信息，当然这些习惯也是卓越的网站培训和引导的。在这个习惯和社群理念中，其它的第三方平台想要改变、转移他们的价值观是很难的，这就是我们的社区化经营的最高界境界。

该模型集合我们华夏互联多年来服务行业客户的经验，有较好的参考价值。

在这个模型的引导下，我们有过成功的实施，比如为北京某人才网站，成功的定位为&ldquo;找工作、学技能、同业交流&rdquo;三个功能，通过后学技能、同业交流两个新增功能，服务于找工作的第一功能，忽略传统的企业推介，专注服务于求职者一个群体，有的放矢，取得了卓越的成功。

 --摘自《面向社区与习惯匹配的人才网站发展战略研究》

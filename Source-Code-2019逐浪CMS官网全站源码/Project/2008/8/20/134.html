本案实施细节与亮点披露

1、 内核系统
逐浪CMS采用Microsoft .net2.0架构，基于Microsoft Visual Studio 2005平台开发。微软公司的.net策略实施以来，经过多年的耕耘，取得了卓越的成果。其成熟的商业化应用，不仅可以减少最终用户的商业支出，更重要的是能够提高用户的效率。
数据库内核采用Microsoft SQL Server2000平台，同时兼容包括Microsoft SQL2005和最新发布的Microsoft SQL2008。系统内置了扩展接口，在未来的版本中，还将支持包括DB2、Oracle等数据库，甚至包括Mysql这样的开源数据库。
而外在表现层完全采用W3C标准规范来构建，完全支持智能导入导出操作，为用户提供了自由便利。
逐浪CMS内核采用多层开发，使得庞大的开发工程，在后期维护中非常简单便捷易用。模块化设计保证了项目的运行效率。
2、 数据计算
有一个故事，说的是中国某电信公司的老总某次澡，结果澡没洗好，一路上都是收费，处处设卡，如果软件开发也是这样，那显然是没有前途的。
作为通用软件，商业化是其生存的基本条件，但软件开发者需要明确的是，所有的一切设计都应该以造福用户为己任，而不是完全以收费为目的。
当前CMS行业有一些厂商，完全以设计化开发设计为目的，结果造成整个程序运行效率低。非人性的收费模块设置，造成用户使用的层层受阻，并且受恶意广告的干扰，严重影响站点功能与计算速度。这就好比一个武功高手时时想着独霸武林，而真正的武术是学无止境的，只有专注于技术的侠者，才能领会真正的武学精华。
逐浪CMS从设计之初即确定了“以功能、效率和运算来打动用户”的发展理念，摒弃了华丽的外观、粗糙的外表、为商业收费而设置的非人性脚本干扰、商业化防破解设置等，完全面向功能设计，确保程序运行的效率，完全面向客户需求设计，以保证CMS的应用效能。
随着处理器内核的不断发展，CMS应用将朝着矩阵式服务器的趋势进化。在分布式运算中，如何处理好各个阵列的数据交互、功能切换、API接口间的对话，将是CMS应用的关键功能。逐浪CMS利用华夏互联（www.hx008.cn）的矩阵服务器优势和HP服务器合作伙伴的资质，进行了丰富的测试与研发，为CMS的云计算打下了良好的基础，带领中国CMS开发进入“云”时代。
不少CMS强调自身功能强大，但对硬件要求高，即使在通用环境下也不能正常使用，而需要为其配备专门的大型服务器来使用，甚至于后台响应时间慢、过多的JavaScript调用，造成计算效率低，甚至于暴露出安全隐患。

3、 表现层兼容
独创中文标题作为生成格式提供了卓越的SEO功能
对于大型门户与推广型网站而言，搜索引擎优化（SEO）质量决定了一个网站的生存能力，逐浪CMS支持面向HTM、HTML、SHTML、SHTM的静态生成发布功能，支持指定栏目目录，使站点结构更加优化，方便搜索引擎收录。
同时，独创的“中文标题+后缀”生成的形式，更能为SEO优化提供卓越的优化效果。如果当你某天访问到一条http://www.zoomla.cn/news/2008年特价商品热卖.html时，请不必惊讶，这正是源自Zoomla! CMS的卓越贡献。
4、 人性化访问
作为网站，浏览器是将站点内容传递到用户表现层的重要管道，CMS系统对浏览器的兼容性一定程序上可以反映出该CMS的可用性与应用能力。
作为业内浏览器市场的事实主宰者，微软公司IE6发布近三年没有一次更新，而IE7.0版本浏览器发布不到半年，就发布了IE8 beta版，由此足见IE8策略的重要。
逐浪CMS系统从一开始就确定了高起点的研发策略，并以IE8为研发标准，同时确保兼容Firefox等主流浏览器。
兼容最新标准的CMS，从根本上杜绝了不少客户一年一改版以适应新版浏览器规则的烦恼。完全支持W3C标准为中国网站的标准之路铺垫基础
中国走向世界、世界了解中国。
未来的网络化传播之道，国际化是其重要趋势，信息时代的文明终将造就地球村内的每一处居民便捷的交互信息。
逐浪CMS系统采用国际标准UTF8语言作为表现层核心语言，从而使网站能够轻松应变国际，避免在不同的语言环境下的编码冲突，在任何国家、地域都能正确呈现网站内容！
5、 个性化应用
逐浪CMS系统经过近一年的开发，总投入超过近百元人民币的研发资金，此次发布的逐浪CMS系统包括独有的自定义模型与节点功能、独创的项目管理系统（Zoomla! Projects）为传统B2B企业提供了卓越的功能支持、独创中文标题作为生成格式提供了卓越的SEO功能、集成逐浪SNS等多项专有技术、独有的在线纠错功能，经过业内专业人士测评后，均给予高度赞赏和肯定。
独创的项目管理系统（Zoomla! Projects）为传统B2B企业提供了卓越的功能支持
未来中国互联网市场中坚在哪里？中小企业和传统生产、贸易、实业型企业显然是重要的一枝力量。普通的CMS系统提供的“网上商城”型应用，显然不能满足这类企业的需求，因为我们的客户不仅仅是将实物产品通过网络商城卖出去这么简单，更多的形式是以诸如订单订制、协同开发、流程委托、研发外包、过程监控等形式进行商业的运营。
逐浪CMS内建的项目管理系统(Zoomla! Projects)，专门为传统的实业型、研发型、贸易型和精密协同开发型企业设计，用户可以自由的通过后台建立项目、初始化项目进程、定义项目节点权限、管理项目满意度、并编辑丰富翔实的项目流程与过程监控体系，通过丰富的项目流来管理项目，基于国际互联网的平台交互，为传统企业插上腾飞的翅膀。
6、 集成逐浪SNS，为社区化运营铺就通途
SNS（人际关系网络）是WEB2.0最重要的核心理念，它揭示了商业与人际关系网络产生关系的核心。逐浪SNS开发团队从2007年组建，到目前为止已经推出了其第一个版本，伴随逐浪CMS1.0的发布，已经能够做到无缝整合，让传统的网站不再是一个信息孤岛，而是用户时时访问的社区型平台，让访客成为真正的流量与效益。
7、 二次开发支持
逐浪CMS在业内首次提出了“箱子”理念，即所有的应用都是可拆卸的，不同的用户可以打开“箱子”的不同展面，以满足其应用。
其于卓越开发平台研发的逐浪CMS，为适应未来站点发展的需要，独创的模型设计理念与无限节点功能，用户可以根据自己的网站需要建立各类模型，从而满足个性化开发的需要，不再限于传统的“内容管理系统”只能简单的进行界面修改和设计而已。
开箱即得的设计理念，源于逐浪软件开发组对用户和市场的深刻研究认知。逐浪CMS前期通过近万名用户数据的调查分析，发现大多数CMS开发者都把其应用者归属为同一类型的用户，从而使研发出来的产品出来缺少灵魂，即使是众口难调，但为每一个层级、不同应用需求的用户提供相应的支持，则是必不可少的。于是，逐浪首次提出了“开箱即得”的CMS开发理念，让不同的用户都能在逐浪CMS的应用与二次开发上找到准确的座标。
逐浪CMS内置自主开发的模板引擎，支持用户自由定义界面、风格与设计，采用XML编程，大大加快了计算速度，模板与程序分离，无论是安全性还是易用性上都是先人一筹。
为了更好的对外部数据进行对接，逐浪CMS同时开放API接口，任何符合逐浪CMS用户API接口规范的外部程序（如论坛、博客），都可以自由整合，也可随时定义开放整合，或关闭整合。
8、 更新与技术支持
我在雷德蒙工作时，曾有个兄弟团队的资深软件开发工程师干过这样一件事。产品开发进入修补漏洞的后期，在这个阶段只有重要的漏洞才能去修补，而且代码提交前要经过伙伴测试（Buddy Test），这样做的目的是要保持产品质量的稳定性，可他过于自信了，在没有经过伙伴测试就把代码提交了。结果他的提交使第二天的每日构建（Daily Build）通不过多个重要的测试用例，兄弟团队也没法儿用它进行其它测试，白白浪费了一天时间。
--微软中国研发集团服务器与开发工具事业部部门经理 徐鹏阳

徐鹏阳先生作为中国程序员界的知名人物，其上述案例揭示了安全维护与更新的重要性。作为基于WEB访问的程序，安全性是其首要任务，国内有一些CMS团队开发的CMS安全性严重存在问题，以致于不断更新，甚至于在用户后台置入强制的更新模块、干扰用户操作选择—这样的流氓行为将严重妨碍用户使用。
逐浪CMS从一诞生起，就非常注意安全性与更新支持，我们的更新不采用强制行为，而是采用人性化的推送，通过开发者社区、定向邮件、准告短信和专业工程师上门更新、远程协作等方式进行安全维护，注重用户的使用体验和隐私保证，构建专业的安全防护体系。
网络上危机无处不在，作为企业形象的网站，在运营过程中会遇到各种安全问题，如：
•黑客会通过各种SQL注入漏洞对网站进行攻击。
•黑客通过文件和图片上传功能上传一些木马程序取得网站甚至服务器的控制权。
•黑客可能会采集恶意刷新某些页面使服务器当机。
•黑客可能会通过暴力破解或Cookie欺骗等手段来获取管理员的密码，从而达到控制网站的目的。
逐浪CMS的解决方案中通过以下的各种安全手段来保证网站的最大安全：
•周密的安全策略和攻击防护，全面防止各种攻击手段，有效保证网站的安全。系统在用户资料存储和传递中，对密码全部进行 MD5 不可逆加密处理，有效保证系统及用户资料的安全。为脆弱页面专门设计了防刷新系统，自动阻止恶意访问和攻击；安全检查应用于每一处代码中，每个提交到系统查询语句中的变量都经过过滤，可自动屏蔽恶意攻击代码，从而全面防止SQL注入攻击、密码猜解、上传木马等各种攻击手段，最大程度地保证了系统的安全和稳定。独有的全程操作记录功能，完整详细的记录了系统的各种运行情况，让管理员对网站运行情况随时有据可查。即使管理员也无法删除两天内的记录，使得日志记录模块犹如黑匣子般为系统提供最可靠的安全保证。
•逐浪CMS内置安全屏蔽体系，可以屏蔽特定的攻击。系统编写了单独的上传组件，上传文件的黑名单类型达到26种，确保不可能直接上传这些文件。对上传文件的扩展名做了严格限制，只允许26个英文字母＋10个数字。上传至主机中的文件名由系统生成，确保黑客不可能通过各种方式在上传过程中出现非法之类的可执行文件。在利用系统上传功能上传文件后，系统会对当前上传目录进行扫描，一旦发现26个黑名单类型中的文件则立即自动删除，布下了最后一道不可能突破的关口，以保证黑客不能通过上传的方式入侵网站系统。
•系统提供安全的管理认证码设置功能，可使管理员进入后台时除了输入用户名、密码、验证码外，还要输入设置存放在文件中自定义的安全管理认证码。除非攻击者拥有了FTP或者得到WebShell权限（可以通过Web查看、修改、删除服务器上的文件），否则攻击者不可能得到管理认证码，以确保网站系统的安全。
•管理员登录、黑客攻击、越权操作、日常操作等都记录在案，并提供给超级管理员查询。逐浪CMS拥有完善的日志管理机制，除特定管理员删除、清空叁天前的日志外，其它日志谁都不能删除，以方便网站管理员及时通过日志找到问题原因。为防止盗用密码和猜破密码，后台管理员登录界面的密码输入框可选使用软键盘，增强了密码的安全性。
9、 开发者联盟
逐浪软件作为CSDN核心合作伙伴、CSDN外包全球TOP首邀金牌（唯一）合作伙伴，与包括CSDN在内的专业技术平台结为合作，面向专业开发者开放，打造卓越的网站内核管理系统。同时，逐浪CMS采用面向最终用户免费、向开发者开源的策略，所有用户只要填写一个简单的申请表格（未来还将开通在线版本申请表）申请加入“逐浪CMS开发者俱乐部”，即可拥有相应的源码查看与使用权。

　　优势一、独有的自定义模型与节点功能
　　逐浪CMS在业内首次提出了“箱子”理念，即所有的应用都是可拆卸的，不同的用户可以打开“箱子”的不同展面，以满足其应用。
　　其于卓越开发平台研发的逐浪CMS，为适应未来站点发展的需要，独创的模型设计理念与无限节点功能，用户可以根据自己的网站需要建立各类模型，从而满足个性化开发的需要，不再限于传统的“内容管理系统”只能简单的进行界面修改和设计而已。
 
　　优势二：独创的项目管理系统为B2B企业提供卓越功能支持
　　未来中国互联网市场中坚在哪里?中小企业和传统生产、贸易、实业型企业显然是重要的一枝力量。普通的CMS系统提供的“网上商城”型应用，显然不能满足这类企业的需求，因为我们的客户不仅仅是将实物产品通过网络商城卖出去这么简单，更多的形式是以诸如订单订制、协同开发、流程委托、研发外包、过程监控等形式进行商业的运营。
　　逐浪CMS内建的项目管理系统(Zoomla! Projects)，专门为传统的实业型、研发型、贸易型和精密协同开发型企业设计，用户可以自由的通过后台建立项目、初始化项目进程、定义项目节点权限、管理项目满意度、并编辑丰富翔实的项目流程与过程监控体系，通过丰富的项目流来管理项目，基于国际互联网的平台交互，为传统企业插上腾飞的翅膀。
 
　　优势三：独创中文标题生成格式提供卓越SEO功能
　　对于大型门户与推广型网站而言，搜索引擎优化(SEO)质量决定了一个网站的生存能力，逐浪CMS支持面向HTM、HTML、SHTML、SHTM的静态生成发布功能，支持指定栏目目录，使站点结构更加优化，方便搜索引擎收录。
　　同时，独创的“中文标题+后缀”生成的形式，更能为SEO优化提供卓越的优化效果。如果当你某天访问到一条http://www.zoomla.cn/news/2008年特价商品热卖.html时，请不必惊讶，这正是源自Zoomla! CMS的卓越贡献。
 
　　优势四：首个完全支持IE8标准的CMS系统
　　作为网站，浏览器是将站点内容传递到用户表现层的重要管道，CMS系统对浏览器的兼容性一定程序上可以反映出该CMS的可用性与应用能力。
　　作为业内浏览器市场的事实主宰者，微软公司IE6发布近三年没有一次更新，而IE7.0版本浏览器发布不到半年，就发布了IE8 beta版，由此足见IE8策略的重要。
　　逐浪CMS系统从一开始就确定了高起点的研发策略，并以IE8为研发标准，同时确保兼容Firefox等主流浏览器。兼容最新标准的CMS，从根本上杜绝了不少客户一年一改版以适应新版浏览器规则的烦恼。
 
　　优势五：完全支持W3C标准为网站标准之路铺垫基础
　　中国走向世界、世界了解中国。未来的网络化传播之道，国际化是其重要趋势，信息时代的文明终将造就地球村内的每一处居民便捷的交互信息。逐浪CMS系统采用国际标准UTF8语言作为表现层核心语言，从而使网站能够轻松应变国际，避免在不同的语言环境下的编码冲突，在任何国家、地域都能正确呈现网站内容!
 
　　优势六：集成逐浪SNS，为社区化运营铺就通途
　　SNS(人际关系网络)是WEB2.0最重要的核心理念，它揭示了商业与人际关系网络产生关系的核心。逐浪SNS开发团队从2007年组建，到目前为止已经推出了其第一个版本，伴随逐浪CMS1.0的发布，已经能够做到无缝整合，让传统的网站不再是一个信息孤岛，而是用户时时访问的社区型平台，让访客成为真正的流量与效益。
 
　　优势七：营销策划一体化的解决方案
　　我们认识到，在本土企业与商业应用环境中，缺少的并非是优秀的程序与技术，而很多时候往往因为整体解决方案与应用层架构的不理想，造成IT投资的极大浪费。
　　逐浪软件携业内专业互联网解决方案服务商华夏互联、中国营销策划界翘楚华夏营销网打造金三角增值值体系，所以成为逐浪CMS商业用户，均有权享受此项服务。
 
　　优势八：独有开发者俱乐部、面向专业开发者提供支持
　　逐浪软件作为CSDN核心合作伙伴、CSDN外包全球TOP首邀金牌(唯一)合作伙伴，与包括CSDN在内的专业技术平台结为合作，面向专业开发者开放，打造卓越的网站内核管理系统。同时，逐浪CMS采用面向最终用户免费、向开发者开源的策略，所有用户只要填写一个简单的申请表格(未来还将开通在线版本申请表)申请加入“逐浪CMS开发者俱乐部”，即可拥有相应的源码查看与使用权。

本案曾实施的案例精选推荐

 


1. 网行贵州
网行贵州是一个大型区域门户网站，站点集新闻、旅游、社区、黄页、交友等为一体，同时结合在线地图功能，站点规则打造成一个中南地区最大的社区服务类门户。项目立项之初，项目甲方将其定位为“集口碑网、我要点评网、饭桶网和芒果网为一体的大型综合合门户”，同时为用户提供论坛、博客、圈子等功能。
逐浪项目组承接此项目后，确定了“高标准、高起点、高运算效率”的三高开发理念，采用工厂开发模式和多层架构开发，经过近半年的研发，成功完成了项目，赢得了客户的好评，并成为当地网友首选的地方门户。
2. 我要队伍网
我要队伍网是由北京恒言公司委托逐浪CMS团队开发的网游类门户，站点主要为游戏一族提供帐号交易、游戏点卡交易、队伍交流的门户网站。
站点除了传统的商业贸易功能外，同时还集成了短信、SNS等功能，为客户商业赢利提供了夯实的基础。该站点采用逐浪CMS开发团队最新研发成果，集成了SNS、在线交易、大型信息发布系统、论坛系统、博客系统、在线充值系统，成为了年轻一族热捧的时尚门户。
3. 3K壁纸网
3K壁纸网是基于图片共享的大型门户，用户可以自由上传图片、换取积分、自由交流。站点要求可以实现用户上传图片、断点续传、自动打包、根据会员级别的提供下载权限，访客可自由评论，用户间无缝的交流，系统并根据这套评论系统可对用户上传资料进行评级、打分，从而提高上传资源的利用效率。
这样的站点对于数据交互、服务器性能、优化计算、客户端与服务器数据流处理都有较高要求，而逐浪CMS本身就具备会员级别、角色设计、上传管理等诸多功能，客户导入逐浪CMS的成熟开发，不仅成本得到了降低，更重要的是缩短了开发成本，赢得了宝贵的市场商机。
4. 摇头DJ网
摇头DJ网是ALEXA全球排名10万的娱乐门户，在接触逐浪CMS项目前，站点运行效率低、速度慢、服务器经常宕机—摇头DJ网这种案例是较具普遍性的，不少客户都有过IT投资、并建有网站，在采用逐浪CMS前都会担心诸如之前开发是否会前功尽弃、是否会有导入困难的问题。
虽然摇头DJ网旧版也是采用.net架构开发，但由于其技术内核其于是.net1.1，不能符合当前的主流应用，同时没有正规的管理后台、模板管理、界面开发、节点与模型等功能，维护成本巨大，并缺乏灵活性。导入正规的CMS开发对其来说显得尤为重要。
逐浪CMS为摇头DJ网导入了平滑升级方案，采用了自主数据迁移程序，将原有数据平滑升级到逐浪CMS商业版，并融入AJAX开发、优化计算（升级后的运行效率比旧平台提升了近200%）。同时，逐浪CMS新版人性化的后台、专业的管理工具，为项目管理提供了强有力的支持，网站运营如虎添翼，为广大DJ爱好者提供了更好的应用。

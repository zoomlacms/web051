解决方案背景

 

2007年1月17日经国务院第165次常务会议审议通过、并于2007年4月5日经温家宝总理签署颁布、2008年5月1日开始施行的《政府信息公开条例》明确规定“各级人民政府应当加强对政府信息公开工作的组织领导”。
目前各地政府、各部门正在积极的开展政府上网、电子政务与信息公开相关内容的规划与建设。作为国内领先的政府门户、电子政务信息化建设的领导厂商，Zoomla!逐浪CMS技根据各级政府部门对政务公开工作信息化的需要，结合政务公开在政府部门中的实际工作方式与流程，适时推出了“Zoomla!逐浪CMS”，为用户提供一流的政府门户、信息公开、商务办公应用平台。

本案实施细节与亮点披露

 

Zoomla!逐浪CMS是基于微软公司ASP.NET2.0构架设计，其低层语言采用了最成熟的C#语言，作为英特尔软件开发合作伙伴，我们拥有鼎级的程序构建与设计师，确保为客户提供最优质的解决方案与产品服务。
高端数据库支持：
Zoomla!逐浪CMS提供MSSQL与Oracle两个版本的数据库服务，普通客户采用MSSQL即可胜任大容量的信息存储服务。
对于大型应用、特殊领域（如军事），我们则提供了Oracle版高端数据库网站管理系统服务，满足特殊应用的需求，并为超海量访问与交互提供了基础，这在国内的CMS厂商中也是屈指可数的。
性能扩展支持：
Zoomla!逐浪CMS政府门户解决方案具备八大自主功能，独有的自由模型、无限节点、超级项目管理、中文名生成支持、UTF8语境与W3C支持—所有的一切，都构成了灵活运用、高性交互的基础。
易用性：
系统采用自主模板引擎，具备“所见即所得”的功能，使用者只需要具备普通的OFFICE办公技巧、掌握30%的WORD软件操作技能，就能灵活的运用网站管理系统。
网站采用符合W3C的语法规则，完全支持UFT8国际语言，使网站的易用性、交互性得到了保障。
安全保障
Zoomla!逐浪CMS采用的N+级安全开发理念。
首先在开发之初，就部署了高级的安全配置，并始终贯穿在开发过程中。
Zoomla!逐浪CMS的开发者拥有十年IDC行业经验，具备大型SERVER管理与开发能力，我们首创将云计算的安全技术转移到了CMS的开发上，保证了系统的安全性。
其次，Zoomla!逐浪CMS作为最受欢迎的CMS，多年来一直深受网友喜爱。Zoomla!逐浪CMS开发团队建立了与各大开发社区的密切合作关系。广大的用户是我们安全的保障，动态跟进服务，改进产品，铸造一流的应用。
更新服务
Zoomla!逐浪CMS平均每年发布一个版本，每个月都会有功能更新。根据客户的应用反馈，Zoomla!逐浪CMS项目会提升原有功能、开发新功能，满足政务系统的应用需求。
我们深知，这个世界唯一不变的只有“变”，唯有创新，才能为包括政府在内的诸多客户提供卓越的服务。
一体化服务
Zoomla!逐浪CMS的主要目标是通过信息化手段加强政府信息公开工作的建设和管理。按照各地、各行政机关此项工作的具体要求，通过系统平台可以轻松实现网站内容高度整合、表现方式统一、资源全面共享，从源头上保障信息公开的规范、有效和及时，形成上下同步、规范统一的政府信息公开监管体系。通过信息目录体系与依申请公开实现主动公开与依申请公开的政务信息公开方式，保障公民、法人和其他组织依法获取政府信息，提高政府工作的透明度，促进依法行政，充分发挥政府信息对人民群众生产、生活和经济社会活动的服务作用。

Zoomla!逐浪CMS以成熟的技术，用于快速构建电子政务信息公开体系，产品提供了目录管理、信息管理、模板样式管理、维护体系管理、依申请公开办理等公开流程所需的各项功能，系统支持横向、纵向的单位构架体系，实现了内外网、多地域、多部门的信息交换和报送；同时完善的维护权限管理，确保了系统的规模型、可靠性、安全性。

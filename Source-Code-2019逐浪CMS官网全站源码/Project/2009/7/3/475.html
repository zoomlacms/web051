解决方案背景

 方案全称:华夏互联之逐浪CMS在线试衣与色彩模块FLEX以及基于ajax的3D应用解决方案

解决方案目的：
通过本方案的实施，重点在于实现虚拟的场影与现实紧密关联，同时进行实时的动作操作，以达到与实际运营相匹配的目的。

3D的应用：
基于ajax的开发，实现以虚拟展台形式的现场演示效果，用户如同游场景和第二人生一样的平台。
用户一旦登陆后，可以实时在街区行走，并进行会话，可以发起一人或多人的实时聊天。用户可以畅游各个街区，如果程序应用方需要的话，还可以进行对街道进行分区，形成一个如同实景一样的交易现场。
街区的每个店铺的广告都可以实时的更新，为商家提供最优秀的展位。

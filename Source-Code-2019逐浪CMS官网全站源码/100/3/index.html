南昌珠宝检测网

注册

国土资源部南昌矿产资源监督检测中心隶属江西省地矿局地勘院，江西省最早的、最权威的金银珠宝产品检测单位；于1991年成立江西省矿产品质量监督检验站（赣质监认字011号），承担全省矿产品及黄金珠宝首饰等六大项检测，已有二十多年的历史，于2006年通过了国家级计量认证，并成立了国土资源部南昌矿产资源监督检测中心。
本中心具有先进的检测设备、专业检测人员和严密的质量保证体系。拥有多台珠宝显微镜、万分之一电子天平、大型X荧光光谱仪、激光光谱仪等先进的黄金、珠宝检测仪器设备。可开展钻石、红宝石、蓝宝石、翡翠等各种珠宝玉石品种和金、银、铂、钯饰品的无损检测。 本中心各种矿产品检测人员80多人，从事珠宝玉石鉴定人员6人，专门从事各种珠宝玉石和金、银、铂饰品的无损检测 。其中工程师和高级工程师占总人数的80%，国家珠宝玉石注册检验师2人、宝协珠宝鉴定师1人。所有检测人员每年参加专业知识培训、再教育、学术研讨会及交流会，及时吸取和接受新知识，不断增强自身检测能力。

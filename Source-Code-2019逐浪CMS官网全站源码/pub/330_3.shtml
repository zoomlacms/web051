详细说明
编者按
自由的网络世界，给我们带来的不仅是单纯的信息，更因其开放、自由，不受空间、时间的影响，为我们带来了全新的生意网络。
然而长期以来，普通网民离电子商务的距离依然很远，除了简单的在淘宝、拍拍、百度有阿等门户开一个免费小店外，要自己建站简单是天方夜谭。
虽然淘宝、拍拍、百度有阿这些大型门户提供了免费开店的功能，然而想在这些平台上经营成功岂是一朝一夕所能达成？这些网店一方面拥有海量的商家，商家间竞争非常激烈，没有一定的实力和卓越的性价比很难吸引买家的眼球；另一方面这些平台都开僻了专业的品牌专区和商城，而普通的中小商家很难在竞争中取胜。
面对越来越激烈的竞争格局，大多数企业都选择了独立网站营销之路，通过独立的网络营销建立起了权威的影响力，取得了卓越的成功。
作为国内领先的CMS与网店解决方案软件开发厂商，逐浪软件从一开始就定位于为广大企业、站长、个人提供卓越的电子商务门户网络平台，在其即将推出的2.0版本中，商务的功能将更趋强大，让我们一同来体验其卓越的网店功能。
销售、团购、竞拍统统搞定
传统的网店程序，只能简单的发布商品销售，而流行的竞拍、团购等功能都无法实现。
在新版的逐浪CMS网店中，集成了全新的商品发布机制，用户可以自由的发布正规商品、团购或竞拍。
Zoomla!逐浪CMS2.0版架构师对此的描述是：为枯燥的商业销售融入创新的WEB2.0元素，以满足新商业发展趋势的需求。

高品质的商品管理，让应用随心所欲
在全新的逐浪CMS2.0版本的网店功能中，采用全新的AJAX开发，用户可以随心所欲的建立起功能强大的商品发布管理体系，即使是上万条信息分类条目，也能轻松搞定。
而为了满足B2C、C2C用户的高性能应用需求，系统采用智能优化后的算法，即使是上万条数据，后台更新其中一条，前台也可以轻易实现无刷新异步传输，满足大型商城运营的发展需求。

灵便应用、整合强大的商业应用体系
作为网站内核厂商，Zoomla!逐浪CMS的目标并不仅仅是以提供一个商城给客户为目的，而是为传统网站植入高品质的电子商务应用流程，构建起功能强大的企业信息中心、商务中心、品牌中心、管理中心以及电子文档管理中心。
其中使用逐浪CMS网店系统，可以在原有的商城基础上，整合行业商务体系。

详细说明

逐浪SNS专注于作社区运营的领先者。

SNS，全称Social Networking Services，即社会性网络服务，专指旨在帮助人们建立社会性网络的互联网应用服务。也指社会现有已成熟普及的信息载体，如短信SMS服务。SNS的另一种常用解释：全称Social Network Site，即“社交网站”或“社交网”。

 Zoomla!逐浪CMS是国内高端CMS代表，其SNS功能集成于员中心应用，安装好Zoomla!逐浪CMS后，只要点击用户登陆，输入初始用户名：admin，密码：admin888，登陆会员中心或直接访问http://您的网址/user/Default.aspx即可进入会员中心。
进入会员中心，点击我的空间，即可进入SNS初始化界面，如下图所示：

Zoomla!逐浪SNS集成了婚介、好友、聊天、读书、游戏、蒙面聊、旅游、电影、抢车位、斗牛、公会等模块，其与CMS系统高度结合，可以用于快速构建起卓越的社区网站。

每一个会员在SNS中都拥有唯一的ID标识，并可以建立起自己的个人主页、用户收藏、用户点评、会员短信、生日提醒等模块，应用卓越。

1967年，哈佛大学的心理学教授Stanley Milgram(1934～1984)创立了六度分割理论，简单地说：“你和任何一个陌生人之间所间隔的人不会超过六个，也就是说，最多通过六个人你就能够认识任何一个陌生人。”按照六度分割理论，每个个体的社交圈都不断放大，最后成为一个大型网络。

而逐浪SNS则根据中国本土国情的不同，进行本土化理念与功能的深度研究，从而使之更加适合于华人社区使用。

其卓越的可二次开发性，也是保证逐浪SNS在高端社区门户中受欢迎的原因所在，用户还可以自由的根据自己的需要进行再次开发，开放的计算能力保证了系统的卓越功能。

详细说明

网络安全的世界，就是信息安全的底限。

伟大的雅虎公司程序员发明了验证码，但随着人工智能和机器识别的提升，道高一尺、魔高一丈，传统的验证码已经满足不了网站和web平台的安全需要，于是我们在全新的Zoomla！逐浪CMS2 x3.9.3版（即将全面发布）中，提供了全新的点触验证码。

这是逐浪软件与点触科技强强合作的结晶。

点触验证码是颠覆传统字符验证码的新一代验证码系统。用户通过点击、拖动等有趣方式即可完成验证,防止机器攻击, 应用在注册、登录、数据访问保护、黄牛刷单等场景。

只要在逐浪CMS后台-系统-配置 ，进行简单的配置，就能快速集成与京东、腾讯、阿里一样的高科技验证码，从而避免恶意注册、破解，提升平台安全性。

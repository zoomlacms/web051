详细说明

曾经有一个理想：网站开发能否不进后台，随心所得，跟PS、WORD一样，写到哪里，哪里就是前台？

曾经有一个梦想，能否更加敏捷的管理网站，没有后台，只有中台。


今天，随着Zoomla!逐浪CMS2 x3.9的神秘面纱逐渐张开，其“超级按钮”功能，正款款走来，带来了全新的用户体验、革命性的管理应用，提升站点管理和接受能力，易用性提升3倍。

是的，我们没有休息，在2016年春节之际，为所有的商业用户保障服务，提供了这一全新商业功能，并将在下一个版本发布中，全面公开。


通过这一功能，用户可以在任何的前台页面，根据自己需要，点击“超级按钮”，激活相应菜单，包括对此栏目添加信息、维护此栏目页面样式、快速进入后台、管理员信息，而不需要进入后台避免由于技术知识不足的缺陷。

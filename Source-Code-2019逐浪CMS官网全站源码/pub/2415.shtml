详细说明



长期以来，企业级用户都希望对自己的信息管理拥有强劲的工具，可以按需订制，在浩瀚的互联网领域中，管理完整的信息，并随时随地按需订制自己的舆情重点。

今天，Zoomla!逐浪CMS带来了其年度企业级信息化管理工具—舆情管理系统，助力企业信息化管理。这是逐浪软件人，集10年的软件研发成果，融入大数据、云计算、企业级应用智慧，并在服务广大企业客户的基础上的科研结晶。

五大功能模块，超值值得应用

一、实时监控，把握舆情动向


Zoomla!逐浪CMS舆情监测系统，可以自由订制关键字、关键词、来源、采集目标。

比如针对高考、微软、互联网、家暴等关键字进行定义，并选择关注的信息源，包括微信、微博以及传统的新闻搜索（如百度新闻、360搜索等搜索来源），从而建立起良好的信息来源，第一时间把握舆论动向，为党政机关、传统媒体、国企、现代集团企业、跨国公司 、军队、报社等领域提供准确的舆情监控服务。

与逐浪CMS紧密结合的好处

一方面体现在强大的二次开发上，对接政力办公。

二方面体面在与移动化应用的完美结合，逐浪软件全线产品整合Bootstrap和HTML5应用，能够高效的应用在微信、手机、微博等平台。

三方面可以与逐浪CMS中的短信通、邮件通等模块结合，支持根据舆情进行短信投递、邮件传送，信息直达决策中枢！


 

 

二、生成报表，支持智能建管理

传统的舆情管理，要么囿于客户端不能方便的管理，另一方面在网页端又没有强大的数据能力支持（比如低端的PHP仿站源码在基础型MYSQL数据上并不支撑大数据）。

Zoomla!逐浪舆情系统具备多年的企业级软件开发经验，在windows server2012R2以及诸多大型机上进行虚拟化部署，并支持将文档生成MHT、DOCX、PDF等不可变文档格式，可以存档，通过云端数据共享，更有二进制数据入库技术，使数据化整为零，只有信息管理员和决策者，才能进行读取。

报表支持复杂的逻辑，包括标准柱状图、条形图、堆积图、折线图、面向图、饼图、环形图、南丁格尔玫瑰图、气泡图、漏斗图等多种模式，使数据管理和舆情监测更加智能。


  

 

三、智能采集，一健入库

系统支持智能的数据采集源，包括百度、搜狗、微信、微博、360搜索等多个搜索源，智能分析数据。

对于分析到数据，如果感兴趣，可以快速一键的导入到文章系统或其它频道（栏目），加快信息采集源。

加上Zoomla!逐浪CMS固有的功能模块，与逐浪软件完美结合，为开发者提供高效的采集信息方法。

无怪懂小姐说：这真是一举两用，既监控了舆情，又采集了信息，编辑与舆情员一人做，工资双份拿，你说任性不任性？

 

 

四、报表丰富，深度挖掘有价值的内容与数据

 考量一个舆情系统的价值，不仅在于其易用性、安全性、稳定性，更在于其对于商业BI领域的卓越能力。

逐浪软件提供一系列的事后分析工具，包括党情舆情评测系统、报表生成系统、投票分析系统、微舆情接入管理，使这一系列工具，使报表丰富、翔实，支持实时演示和实时提示。

 

五、联接微博、微信，助力中国互联网+

众所周知逐浪CMS作为国内网站和Html5领域的领域者，产品完全打通了微博、微信和微应用以及大APP间的脉胳，软件之禅，产品开发（舆情系统目前暂未全开放）度领先同行，不论是否收费版，还是免费版，都可以从更新中，获得相应的应用，为中国互联网+贡献能力。

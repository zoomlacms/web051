详细说明


编者按：

众人期待、群星捧月的Zoomla!逐浪CMS4.6版已经进入了严密的后期开发阶段。据了解，此次版本的更新是最近间隔时间最长 的一个发布，也是2010年上半年最重要的发布里程碑。

本着“专注核心功能、超越应用需求”的理念，Zoomla!逐浪CMS4.6版将朝更易用、易敏捷、更智能的方向 发展，本期开始，我们将逐期进行预告，以展期新版的核心功能。


    众所周知，Zoomla！逐浪CMS一直以为用户带来卓越应用体验为目标，其功能版本也一直秉承单 一版本、统一发布的原则。这样的好处是用户不需要去识别多个版本，也不需要花费很大的人力投资去识别、更不需要为不同的版本间升级迁移花费心血。

    然而，由于功能过多，一些个人长站和普通用户反馈：我并不需要这么 多功能，您提供给我这么复杂、这么强大的系统，给我带来的不是应用上的改进，而是使用上更复杂 了。

    Zoomla！ 逐浪CMS官方团队捕捉到用户的这一个信息，立即对系统的后台布局进行了大幅的改进，同步推出了“百变桌面(Manger style)”的功能。


Zoomla！逐浪CMS老用户可能对4.5版的后台进入桌面提示记忆犹 新，这是一组大幅图片，展示Zoomla！逐浪CMS的入门使用技巧与引导，然而为了使用户获得更好的体验，官方不惜 撤除引导功能，取而代之的是百变桌面功能。在后台的操作台上，用户可以选择不同的桌面模式：

全部显示：显示所有的界面，适合超级管理员使用。
内容管理：只显示内容管理和审核界面， 适合普通编辑使用。
商城管理：管理商订单、购物车以及物流界面，适合面向B2B、B2C开展电子商务的企业应用。
黄页系统：显示企业 黄页和用户管理界面，适合企业用户使用。
社区运营：显示社区工会管理界面，适合论坛工会等功能的应用。
智能办公： 只显示用户管理界面，适合智能办公用户的使用。
配置场景：显示系统配置界面，适用于配置系统参数的管理员使用。
安全 维护：显示配置系统安全界面，适用于维护网站安全的管理员使用。
